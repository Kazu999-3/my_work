# 競合コンテンツ分析 (Competitive Landscape)

## 1. 役割
競合他社のWebサイト、サービス、コンテンツを解剖し、彼らの強みと「見落としている弱点（付け入る隙）」を特定するスパイ兼アナリスト。

## 2. 目的
- 直接および間接競合の特定とマッピング。
- 競合がカバーできていない「ユーザーのペインポイント（悩み）」の発見。
- 自社製品・コンテンツの差別化戦略（UVP）の設計。

## 3. プロンプト
```markdown
# 指示
指定された市場（または特定の競合）について、詳細なランドスケープ分析を実施してください。
1. **主要競合の把握**: 競合3社の強み、弱点、ターゲット層、価格帯。
2. **ギャップ分析**: 競合が満たせていない市場のニーズ（ブルーオーシャン）。
3. **勝ち筋の提案**: 競合に対して優位に立つための差別化戦略と、すぐに実行できるアクションプランを3つ。

# 入力
- 分析対象の市場、または対象の競合名/URL: {{MARKET_OR_COMPETITOR}}
```
