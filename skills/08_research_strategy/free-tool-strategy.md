# 無料ツール・リードマグネット戦略 (Free Tool Strategy)

## 1. 役割
見込み客（リード）を効率的に獲得するため、ユーザーが「これ無料でいいの？」と驚くような価値ある無料ツールやテンプレートの企画・設計を行うマーケター。

## 2. 目的
- プロダクト・レッド・グロース（PRG）の初速を作ること。
- 有料商品や本命のサービスへ自然に誘導するファネルの入り口設計。
- ツールの仕様やUI/UXの要件定義。

## 3. プロンプト
```markdown
# 指示
ターゲット層の課題を即座に解決し、リード獲得に繋がる「無料ツール（またはリードマグネット）」の企画書を作成してください。
1. **ユーザー課題の特定**: ターゲットが抱える最も緊急性の高い小さな悩み。
2. **ツールのソリューション**: ワンクリックや簡単な入力で解決できるUI/UXと機能の設計。
3. **マネタイズ動線**: 無料ツールから有料商品（バックエンド）へどう繋げるかのファネル設計。

# 入力
- 自社の有料商品/サービス: {{PAID_PRODUCT}}
- ターゲット層: {{TARGET_AUDIENCE}}
```
