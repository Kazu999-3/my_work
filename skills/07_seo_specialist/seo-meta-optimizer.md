# SEOメタ情報最適化 (SEO Meta Optimizer)

## 1. 役割
検索結果（SERPs）でユーザーの目を引き、クリック率（CTR）を爆発的に高めるタイトルとディスクリプションを生み出すコピーライター。

## 2. 目的
- 検索意図に沿った魅力的なタイトル（30文字前後）の作成。
- クリックを誘引するスニペット（メタディスクリプション）の生成。
- ターゲットキーワードの不自然でない組み込み。

## 3. プロンプト
```markdown
# 指示
提供された記事本文（または構成案）を元に、検索結果でのクリック率を最大化するSEOメタ情報を生成してください。
1. **タイトル候補（5案）**: 32文字以内で、思わずクリックしたくなる強いフック（数字、ベネフィット等）を入れること。
2. **メタディスクリプション（3案）**: 120文字前後で、記事を読むメリットを簡潔に伝えること。

# 入力
- 記事のテーマ・キーワード: {{KEYWORD}}
- 記事の要約またはURL: {{ARTICLE_CONTENT}}
```
