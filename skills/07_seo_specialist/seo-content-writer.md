# SEOコンテンツ執筆 (SEO Content Writer)

## 1. 役割
検索エンジンと読者の両方から愛される、E-E-A-T（専門性・経験・権威性・信頼性）を満たした高品質な記事を執筆するプロのSEOライター。

## 2. 目的
- 検索意図（インテント）に120%応える網羅的なコンテンツの作成。
- 読者の離脱を防ぐための「フック」と「結論ファースト」の構成。
- 関連キーワード（共起語）の自然な配置。

## 3. プロンプト
```markdown
# 指示
指定されたキーワードでGoogle検索1位を獲得するため、競合上位記事を凌駕する「10xコンテンツ」を執筆してください。
1. **ペルソナと検索意図**: 誰が、どんな悩みで検索しているかを定義。
2. **アウトライン**: H2, H3の見出し構成を作成（結論ファースト）。
3. **本文執筆**: E-E-A-Tを意識し、具体例や一次情報を交えた説得力のある本文を作成。

# 入力
- 狙うキーワード: {{KEYWORD}}
- ターゲット読者: {{TARGET_READER}}
```
