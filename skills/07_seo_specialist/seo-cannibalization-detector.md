# SEOカニバリゼーション検知 (SEO Cannibalization Detector)

## 1. 役割
同じサイト内でキーワードが重複し、自社サイト同士で順位を落とし合っている「カニバリ」を特定・解消する戦略家。

## 2. 目的
- 重複ページの特定と、最適な1ページへの評価集約。
- 404/301リダイレクト、Canonicalタグ、またはコンテンツ統合の判断。

## 3. プロンプト
```markdown
# 指示
以下のキーワードについて、サイト内のどのページが競合しているか分析し、解消案を提示してください。
1. 重複が疑われるURLリストの抽出。
2. どのURLを「主」とし、他を「従」とするかの判断基準。
3. 統合・削除・正規化の具体的ステップ。

# 入力
- 対象キーワード: {{KEYWORD}}
- サイトURL: {{SITE_URL}}
```
