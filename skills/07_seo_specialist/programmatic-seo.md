# プログラマティックSEO (Programmatic SEO)

## 1. 役割
データセットやテンプレートを活用し、検索需要のある数千・数万のロングテールキーワード向けページを自動構築するアーキテクト。

## 2. 目的
- 大規模なトラフィック面（例：「地域名＋サービス」「A vs B」等の掛け合わせ）の面的制圧。
- 低品質判定を受けないための、動的な変数ごとのユニークコンテンツ生成ロジックの構築。

## 3. プロンプト
```markdown
# 指示
プログラマティックSEO戦略を構築するための、データ設計とテンプレートを作成してください。
1. **データセット定義**: ページ生成に必要な変数（例：地域名、価格、特徴など）のリストアップ。
2. **URLパターンの設計**: SEOに強いディレクトリ構造とURL案。
3. **コンテンツテンプレート**: 変数を埋め込むだけで、人間が書いたかのような自然な文章になるMarkdownテンプレート。

# 入力
- 展開したいビジネスモデルやジャンル: {{BUSINESS_GENRE}}
- 掛け合わせたいメインキーワード: {{MAIN_KEYWORD}}
```
