# X投稿最適化 & リライトスキル

このスキルは、X（Twitter）の投稿を「140文字以内」という厳格な制約の中で、インプレッションとクリック率を最大化させるための分析・改善を行うものです。単発投稿から複数投稿による「ツリー（スレッド）形式」まで幅広く対応します。

## 1. 評価観点（Xアルゴリズム & 心理学）

### ① 0.5秒で指が止まる「フック」があるか（1枚目）
- [ ] 1行目に強い問いかけ、意外な事実、または強い感情語（は？、衝撃、絶望など）があるか。
- [ ] スマホで見入る際に、最初の3〜5文字で興味を引けているか。
- [ ] ツリー形式の場合、2枚目以降を読みたくなる「予告」や「ベネフィット」が示されているか。

### ② スキャン性（読みやすさ）は高いか
- [ ] 適切な改行、空白行が使われ、視覚的に圧迫感がないか。
- [ ] 箇条書きや対比構造（旧vs新）を使い、一瞬で内容が理解できるか。

### ③ 常識破壊・パラダイムシフトの提示
- [ ] 「努力の否定（実働0秒）」「既存ツールの否定（ブラウザAIには戻れない）」など、読者の常識を覆す要素があるか。

### ④ 証拠（エビデンス）の提示
- [ ] 文中に実績、数値、具体的な成果物の存在が示唆されているか。
- [ ] 画像（スクショ）を添えることを前提とした構成になっているか。

### ⑤ 導線とストーリー性（ツリー形式）
- [ ] **連読性**: 各ポストが単独で完結せず、自然に次へ視線を誘導する構成か。
- [ ] **リンク戦略**: リンクを親ポストに貼らず、最終ポストまたはリプライ欄へ物理的に（↓ ↓ ↓など）誘導できているか。
- [ ] **CTAの説得力**: 最後に「リンクを踏まないと損をする」と思わせる理由があるか。

## 2. 出力フォーマット
各観点について、以下の形式で分析してください。

- **現状の弱点**: 文字数の無駄使い、フックの弱さ、導線の不明瞭さなど。
- **改善案**: 140文字（全角）を厳守したリライト案。
  - ツリー形式の場合は、1/n 〜 n/n 形式で出力。
- **文字数チェック**: 各ポストが140文字以内に収まっているかの判定。

## 3. 改善フロー（必須実行）

### ステップ1：分析
上記の評価観点に基づき、現状の下書きを診断します。

### ステップ2：改善版（vx）の出力
分析結果を反映した、140文字以内の改善案を出力してください。

- **制約事項**: 
  - **140文字（全角）厳守**。
  - ツリー形式は全5〜7ポストでストーリーを組むこと。
  - プロンプト臭を一掃し、人間味と断定力のある口語体を採用すること。

## 4. 使用方法
「この内容をX投稿最適化スキルで[単発/ツリー]形式にリライトして」と指示してください。
