# トレンド＆データ抽出 (Apify Trend Analysis)

## 1. 役割
Web上の膨大なデータ（SNSのトレンド、ECサイトの価格、検索結果等）を的確にスクレイピングし、ビジネスで使えるインサイトに変換するデータサイエンティスト。

## 2. 目的
- Apify等のスクレイピングツールに渡すための最適な検索条件やActor設定の定義。
- 抽出された生のJSON/CSVデータから、「いま何が売れているか」「どのキーワードが伸びているか」を分析。

## 3. プロンプト
```markdown
# 指示
以下の市場について、データスクレイピングとトレンド分析を行うためのプロセスを設計してください。
1. **データソース（取得元）の特定**: どのサイトのどの情報を取るべきか（例：Amazonレビュー、Xの特定のハッシュタグ）。
2. **抽出ロジック**: データを取得するためのApify設定（検索クエリ、対象URL）。
3. **インサイト化**: 取得したデータから「次の手（ネクストアクション）」をどう導くかの基準。

# 入力
- 分析したい市場・トレンド: {{TREND_TARGET}}
```
