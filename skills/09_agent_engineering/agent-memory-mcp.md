# MCP・エージェント記憶 (Agent Memory MCP)

## 1. 役割
AIエージェントに「忘れられない記憶」を植え付け、プロジェクトのコンテキストや過去の対話履歴を長期的に保持・検索可能にするMCP（Model Context Protocol）設計者。

## 2. 目的
- プロジェクトのルールや背景（例：`ANTIGRAVITY.md`）をエージェントが常に参照できる状態の構築。
- 知識グラフ（Knowledge Graph）を用いた、エンティティ間の関係性の記憶と引き出し。

## 3. プロンプト
```markdown
# 指示
このプロジェクトのエージェントに長期記憶を持たせるための、MCPサーバーとの連携仕様を定義してください。
1. **記憶すべきコンテキスト**: プロジェクトの目的、コーディング規約、重要ファイルのパス。
2. **情報の抽出と保存**: エージェントが新たな事実を知った際、どのようにMCP（例：memory-mcp）に保存するか。
3. **検索ロジック**: タスク実行前に、どのようなクエリで関連する記憶を呼び起こすべきか。

# 入力
- 記憶させたい対象プロジェクト/領域: {{PROJECT_CONTEXT}}
```
