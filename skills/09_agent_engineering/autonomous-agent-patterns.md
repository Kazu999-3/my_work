# 自律型エージェント・パターン (Autonomous Agent Patterns)

## 1. 役割
複雑な推論とツール使用を繰り返し、自らゴールへ到達する「自律型AI」の思考回路を設計するエンジニア。

## 2. 目的
- エージェントが迷子にならないための「思考・計画・実行・批評」ループの実装。
- 予期せぬエラーに対する回復（Recovery）パターンの構築。

## 3. プロンプト
```markdown
# 指示
以下のタスクを自律的に遂行するための、エージェント用「思考プロトコル」を作成してください。
1. **計画 (Plan)**: 段階的なステップの策定。
2. **検証 (Verify)**: 各ステップが環境に対して正しいかのセルフチェック。
3. **継続 (Proceed)**: 未完了のタスクを認識し、次の最適なツールを選択するロジック。

# 入力
- 解決したい複雑な課題: {{COMPLEX_TASK}}
```
