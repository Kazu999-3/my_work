# 🚀 行動計画 (action_plan.md)

## 1. 優先順位の定義 (Priority)
1. **収益直結型**: 有料noteの核となる自動化スクリプトの開発・検証。
2. **集客ブースト型**: Xでのインプレッションを最大化する論争的コンテンツの生成。
3. **基盤強化型**: 内部知識（KI）の蓄積と、自己進化ワークフローの改善。

## 2. 実効戦術 (Tactics)
- **批判の反転（Alchemy）**: 批判的なリプライに対し、「その視点は鋭い。実は私も以前はそのフェーズにいた」と一度肯定（共感）し、その後「しかし、Antigravityの視座から見ると……」と上位概念へ引き上げることで、批判者を教えを請う側に転換させる。
- **「グレイブスMid」論争の利用**: LoLのマクロ的正解と、プレイヤーの感情を揺さぶるVibeを融合させたnoteの執筆。

## 3. 鉄のルーチン (Daily Ritual)
- **アカウントパワーの底上げ**: 1日 **30〜100件** のリプライを実行。
- **名刺代わりの一撃**: 1日1本、哲学を凝縮した「長文投稿」を投下。
- **AI週次レビュー (Friday Night)**: 金曜夜、その週で最もインプレッションが低かった投稿をAIに分析させ、原因を「論理式」として抽出、翌週のプロンプトに反映する。
