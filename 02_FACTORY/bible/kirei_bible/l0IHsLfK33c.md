# NEW TIER LIST Changes for Jungle 14.20 | What is BROKEN?

## 📌 動画の結論（1行サマリー）
パッチ14.20ではAPジャングラーがリアンドリーやストームサージのナーフにもかかわらず引き続き強力であり、特にファームを重視し高いスケーリングを持つチャンピオンが環境を支配している。

## 🧠 マクロ戦略・ルート・判断基準

### 全体的なメタの傾向とアイテム変更
*   **APジャングラーの優位性**: リアンドリーのナーフがあったものの、APチャンピオンは依然として強力。APチャンピオンはスケーリングが良く、クリア速度も速いため、適切なチーム構成であればADチャンピオンよりも優位に立てる。
*   **リアンドリーの仮面**: ナーフされたが、いまだに非常に強力なアイテムであり、APジャングラーが環境トップである理由の一つ。
*   **ストームサージ**: ナーフはされたが、マジックレジストを積まない相手に対しては依然としてワンショットキルが可能なほど強力。
*   **コレクター**: バフ（100G安価になりパワースパイクが早まった）により、グレイブスの勝率が上昇したが、APチャンプ相手には依然として弱い。

### ジャングルルートとファームの優先順位
*   **ファーム重視のフルクリア**: ブランド、リリア、グウェン、カーサスといった高スケーリングのAPチャンプは、無闇なギャンクを避けてファームを徹底することが勝利への鍵となる。90%以上の成功率が見込めるギャンク以外は、フルクリアを優先し、自身のパワースパイクを早める。
*   **序盤のインベード**: リリアのような高モビリティチャンピオンは、序盤から敵ジャングルへのインベードを積極的に仕掛け、有利を築くことができる。
*   **オブジェクトの優先順位**: ザイラのようなチャンピオンは、アップテンポなプレイとオブジェクト先行が必須。ミスのないプレイが求められる。

### チャンピオンごとのマクロ判断
*   **グウェン (S Tier)**: ほとんどのAPチャンピオンに強く、集団戦で非常に強力。バランスの取れたダメージを持ち、ペナルティを受けにくい。多くのチャンプがナーフされる中でノータッチだったため、相対的に強化された。序盤はAFKファームで、レベル6以降のギャンクが強力。
*   **リリア (S Tier)**: 高いモビリティを持ち、クレンズを持たない相手にはUltで簡単に集団戦を破壊できる。クリアのナーフは体感できないほど。熟練したプレイヤーが使えば非常に強力で、近接チャンピオンに対して特に有利。
*   **カーサス (S Tier)**: 過小評価されているチャンピオン。リアンドリーとの相性が良く、特に低レート帯では「ハーベスト（魂の収穫）」と集団戦のフィエスタで非常に強力。
*   **ヴァイ (S Tier)**: 唯一のS Tier ADジャングラー。テンポが良く、ギャンクも強力。リセットを獲得できれば1v9も可能。ただし、チャンピオンへの理解と集団戦の知識が必要。エンゲージが必要なチーム構成で、APミッドレーナーと組む場合に有効。
*   **ブランド (A Tier)**: 集団戦が非常に強力で、リアンドリーとライアンドリー（Rylai's Crystal Scepter）の組み合わせがOP。クリアが遅くなったため、無謀なギャンクは避け、ファームを重視する。
*   **スカーナー (A Tier)**: パッシブとEがナーフされたが、依然として非常に強力。タンクが必要なチーム構成であればS Tierにもなり得るが、クリア速度が他のAPチャンプに比べて遅いため、テンポの速いゲームではA Tierに留まる。ゲームが遅くコントロールされる展開であればS Tier。
*   **グレイブス (A Tier)**: コレクターのバフにより勝率が上昇。突っ込んでくる近接チャンピオンに対しては非常に強いが、APチャンプ相手には弱い。チーム構成と相手によってピックを判断する。
*   **ニー・ダリー (A Tier)**: セットアップ（CCを持つサポートなど、ノーチラスやレオナ）が必要。セットアップがあれば非常に強力だが、単独ではポテンシャルを発揮しにくい。
*   **エリス (A Tier)**: ストームサージからシャドウフレイム、マジックペネトレーションブーツと繋げばワンタップキルも可能。状況によってはS Tierだが、他の高一貫性チャンプに比べて安定感で劣る。
*   **リー・シン (A Tier)**: 常に強力なチャンピオンだが、テンポが遅く、ファームして適切なタイミングで適切な場所にいることが重要。

## 🗡️ ミクロ・戦闘のコツ

### ガンクのタイミングと実行
*   **ファーム重視チャンプのギャンク**: ブランド、リリア、グウェンなどのAPチャンプは、高確率（90%以上）で成功すると確信できるギャンク以外は避ける。無駄なギャンクはファームロスとなり、スケーリングを妨げる。
*   **リリアのFlash + Ult**: 相手がクレンズを持っていない場合、フラッシュインからのUltはほぼ確実に集団戦を有利に運べる強力なコンボ。
*   **エリスのワンタップコンボ**: ストームサージ → シャドウフレイム → マジックペネトレーションブーツのビルドパスで、柔らかい相手を瞬時にキルする。

### スキルコンボとポジション
*   **ヴァイのリセット活用**: キルを獲得してスキルリセットが発動すれば、連続して敵を追いかけ、1v9のような状況を作り出すことも可能。自身のポジションと敵の位置を把握し、キルを狙う。
*   **ブランドのRylai's + Liandry's**: ライアンドリー（Rylai's）のスロー効果とリアンドリー（Liandry's）の持続ダメージで、敵を長時間燃やし続け、集団戦を有利にする。敵を常にスロー状態に保つことで、味方が追撃しやすくなる。
*   **ザックの集団戦**: クリアは遅いが、集団戦では多くのスペースを生み出すことができる。敵に強力なキャンセルスキルが少ない場合に有効。適切なタイミングで飛び込み、敵をまとめてCCにかける。
*   **ランブルのUlt**: 適切な場所にUltを配置することで、敵の逃げ道を塞ぎ、持続的なダメージで敵のライフを燃やすことができる。1v1でも優位に立てるため、積極的にデュエルを仕掛ける。
*   **ジャーヴァンのUlt**: 不動のターゲット（マークスマンや一部のメイジ）に対してはUltで閉じ込めることで、確実にキルを狙える。ただし、モバイルな相手には効果が薄い。
*   **カ＝ジックスのHubris活用**: ヒューブリスを積んだ場合、柔らかいターゲットに対するキルポテンシャルが非常に高くなる。孤立した敵を狙い、瞬時にキルする。
*   **フィドルスティックスのUlt**: 相手がマジックレジストを積んでいない場合、適切なブッシュからのUltは集団戦を一瞬で破壊する強力なツール。
