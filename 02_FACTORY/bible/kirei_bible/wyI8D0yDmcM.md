# Want to get Master elo? - Apply these fundamentals

## 📌 動画の結論（1行サマリー）
ジャングルにおける立ち回り、特にフルクリアとファイト選択を徹底的に分析し、情報収集と状況判断に基づいて行動することで、Master エロに到達できる。

## 🧠 マクロ戦略・ルート・判断基準

*   **事前評価:** ゲーム開始前に敵の勝利条件を評価し、どのファイトが有利かを予測する。
*   **フルクリアの優先順位:** トップサイドは3v3になりやすく不利な場合が多いので、ボットサイドを優先的にクリアする。トップサイドのキャンプを取られるとゴールドロスになるため、注意が必要。
*   **レーン状況の把握:** ボットレーンがプッシュできるかどうかに応じて、ボット側のクリープウェーブをカプったりしないかを判断する。常にボットレーンの状況を確認し、ワード設置を促す。
*   **オブジェクト優先順位:** ドラゴンやバロンなどのオブジェクトが出現するタイミングを把握し、それに向けてフルクリアを進める。
*   **情報収集の重要性:** ミニマップを常に確認し、敵の位置や動きを把握する（特にミッドとボット）。敵のワード配置状況も考慮に入れる。
*   **レーンプアリティの重視:** ファイトを選択する前に、レーンプアリティを確認する。レーンプアリティが低い場合、ファイトは不利になる可能性が高い。

## 🗡️ ミクロ・戦闘のコツ

*   **敵のレベルとアイテム状況の確認:** ファイトに入る前に、敵チャンピオンのレベルやアイテム状況を把握する。
*   **ガンクのタイミング:** 敵のワード配置状況を確認し、有利なタイミングでガンクを行う。
*   **スキルの使用順序:** 状況に応じてスキルを使用順序を変え、最大限の効果を発揮する。
*   **チームとの連携:** 常にミニマップを確認し、味方との連携を意識して行動する。
*   **ファイト後の行動:** ファイトに勝利した場合でも、安易な追撃は避ける。フルクリアに戻り、次のファイトに備える。

*   "Waves are so OP. Like everything's based off waves." （ウェーブは非常に重要。ほとんどすべての行動はウェーブに基づいて判断する。）
