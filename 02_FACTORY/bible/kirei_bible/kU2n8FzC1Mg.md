# NARNIA Strategy Explained - How To Carry Games As a Jungler

## 📌 動画の結論（1行サマリー）
敵ジャングルのキャンプの湧き時間を徹底的に管理し、常にインベードを仕掛けることで、敵ジャングラーをマップから消し去り、圧倒的なCS差とマッププレッシャーで試合を支配する戦略。

## 🧠 マクロ戦略・ルート・判断基準

### NARNIA戦略の核心
NARNIA戦略とは、敵ジャングルのキャンプの湧き時間を正確に把握し、常にそのキャンプへの再侵入（リインベード）を狙うことで、敵ジャングラーの成長を阻害し、マップコントロールを奪う戦略である。

**基本原則:**
1.  **キャンプタイムの把握**: 敵キャンプをクリアした場合、2分15秒後にリスポーンする。これを常に記憶し、リスポーン時にその場にいることを目指す。
2.  **クアドラント差の創出**: 自分のトップサイド（またはボットサイド）キャンプをクリアした後、すぐさま敵のボットサイド（またはトップサイド）キャンプへインベードする。これにより、自分は2つのクアドラント（自陣1つ＋敵陣1つ）を確保し、敵ジャングラーは1つのクアドラントしか確保できない状況を作る。

**NARNIA戦略を行うべき条件:**
1.  **1v1で勝ちやすいマッチアップ**: 敵ジャングラーに対し、自然に1v1で有利を取れるチャンピオンを使用しているか、キルやリードによって一時的に有利な状況にある場合。
2.  **少なくとも1つのプッシュレーン**: 侵入したいサイドのレーンにプッシュ力がある、または味方レーナーがテンポを掴んでおり、リードしている状況。レーナーに敵のワードをクリアさせ、インベードに同行してもらうことで安全性を高める。
3.  **レーナーがマップ上にいる**: インベード時に味方レーナーがベース中であったり、ベースしたいタイミングではないこと。理想は、ベースから戻ってきたばかりでヘルス・マナが潤沢なレーナーが同行してくれるタイミング。
4.  **良いベースタイミング**: 多額のゴールドを持っている場合は、インベード前にベースしてアイテムを揃える。これにより、インベード後の小規模戦での勝率が格段に向上する。

**NARNIA戦略を行うべきではない条件:**
1.  **1v1マッチアップで負ける場合**: 敵ジャングラーと遭遇した場合に、返り討ちに遭うリスクが高い状況。
2.  **レーンプライオリティがない場合**: 侵入したいサイドのレーンがプッシュされていて、味方レーナーがフォローに来られない状況。これは非常に危険である。
3.  **レーナーがマップ上にいない場合**: 味方レーナーがベース中、あるいはベースしたいタイミングでインベードを試みるのは避ける。次の機会を待つべき。
4.  **大量のゴールドを持っている場合**: 大量のゴールドを所持したままインベードを試みるのは危険である。一度ベースしてアイテムを揃えることで、その後の小規模戦の勝率が大幅に向上する。

**一般原則:**
*   **キャンプの優先順位**: 自分のキャンプが3つ、敵のキャンプが1つ（または2つ）という状況では、自分のキャンプ3つを優先してクリアする。自分のキャンプ3つは敵のキャンプ1つより価値が高い。ただし、敵のキャンプが2つあり、自分が反対側のクアドラントをクリアできる場合は、インベードも考慮する。
*   **オブジェクトの優先順位**: NARNIA戦略中は、オブジェクトに固執しない。オブジェクトに時間を割くと、良いベースタイミングの維持、キャンプのクリア、敵へのリインベードといったNARNIA戦略の核となる動きが阻害されるため。オブジェクトは、非常に価値が高い場合や、圧倒的に有利な状況でのみ取る。
*   **ウィンコンディションサイド**: 理想的には、味方のウィンコンディション（勝利条件）となるレーンがあるサイドでNARNIA戦略を展開する。これにより、そのレーンをさらに有利にし、勝利への道筋を明確にする。

**NARNIA V1とNARNIA V2:**
*   **NARNIA V1**: 自分で敵のキャンプをクリアし、その湧き時間を計ってリインベードを行う基本的な戦略。
*   **NARNIA V2**: 敵がキャンプをクリアするのを視認した（または推測した）場合、自分でクリアしていなくてもその湧き時間を頭の中で計り、計画的にリインベードを仕掛ける高度な戦略。視界がなくとも敵の動きから推測し、有利な状況を作り出す。

## 🗡️ ミクロ・戦闘のコツ

1.  **スマイトの温存**: 必要がない限りスマイトを温存する。敵ジャングラーとの遭遇戦や、他のキャンプを奪う際に役立つ。
2.  **敵ジャングラーを削る（Chunk Him）**: インベード中に敵ジャングラーと遭遇した場合、無理にキルを狙わずとも、スキルを使って彼らを削るだけでも大きなアドバンテージとなる。これにより、敵ジャングラーはベースを強いられたり、次のキャンプクリアが困難になる。
3.  **効率的なガンクタイミング**: 敵ジャングル深くに潜伏していることで、味方レーナーがオーバーエンゲージしたり、敵レーナーが良いトレードを仕掛けてきた際に、すぐに後ろからガンクを仕掛けることができる。また、敵ジャングラーがトップやミッドでプレイしている間に、ボットサイドでダイブを狙うなど、多様な選択肢が生まれる。
4.  **レーナーへのピング術**:
    *   **注意喚起**: まず味方レーナーの「顔にピングを打つ」ことで注意を引く。これにより、彼らがミニオンのラストヒットなどに集中していても、一度こちらに意識を向けさせることができる。
    *   **意図の伝達**: その後、インベードしたい場所やオブジェクトの場所をピングし、行動の理由を伝える。「顔にピング→理由をピング」の順序が重要。
5.  **状況への適応**: チームメイトが完璧に反応しなくても、死んだら自分の責任である。常に状況を読み、時にはインベードを諦めて安全なプレイに切り替えるなど、生存を最優先に考えて適応することが重要。
