# 【Ahri 16.8.1】戦場を支配する「九尾の狐」―― 勝利の方程式

「知略なき武力は、霧の中を歩くが如し。」

戦術家諸君。パッチ16.8.1のAhriは、もはや単なるメイジではない。敵の崩壊を導くための冷徹な制御装置だ。現在のメタにおいて、Ahriを動かすことは盤上のクイーンを操ることに等しい。

統計とプロトコルに基づき、ランク戦を蹂躙するためのタクティカル・ガイドを提示する。

---

## 1. Core Profile
- **Champion**: Ahri
- **Role**: Mid
- **Power Spike**: Mid Game (15-25m)

## 2. Core Identity
「E（チャーム）の命中精度とR（スピリットラッシュ）による孤立した敵の排除」。
Ahriの真価は、敵を強制的にミスへ追い込む「選択の強制」にある。

---

## 3. Timeline Strategy

### Early Game (0-15m)
- **Directive**: Grubs戦への介入による1500gの優位確保。
- **Action**: ウェーブを押し付け、先行してリバーを制圧せよ。Zyraのようなレーン戦特化型に対しては無理なハラスを避け、マナ管理を徹底する。敵JGがGrubsを狙う際、AhriのEがあるだけで敵は迂闊に動けない。これが戦場の「抑止力」だ。

### Mid Game (15-25m)
- **Directive**: Aatroxのような前衛は無視し、孤立したキャリーを仕留める。
- **Action**: 視界スコアを稼ぎ、敵のローテーションを先読みせよ。Ahriの機動力は、サポートが視界を確保する一瞬の隙を突くためにある。Jinxが育ち始める20分以降、彼女を孤立させるか、集団戦の端でチャームを刺せ。勝利への最短ルートはそこにある。

### Late Game (25m+)
- **Directive**: 敵の「エンドゲームプラン」を物理的に崩壊させる。
- **Action**: 40分以降、JinxのGod Tier化を防ぐことは不可能に近い。ならば、攻撃の機会そのものを奪う。Rで敵陣の裏へ回り込み、キャリーを沈黙させよ。1回の成功が、Nexusの陥落を約束する。

---

## 4. Matchup Matrix
- **Favored**: **イモビリティなメイジ全般（Lux, Xerath等）**。Rでスキルショットを無効化し、一方的にキルを回収せよ。
- **Countered**: **高耐久・CC持ち（Galio, Lissandra）**。動きを封じられることは死を意味する。彼らに対しては深追いせず、サイドレーンへのロームで経済差を広げることに注力せよ。

---

## 5. Itemization Logic v2.0

Ahriのアイテムビルドは「マナ効率」と「バーストの閾値」のみを追求する。

1. **Core Rule**: 最初の帰還で「ロストチャプター」を確保すること。マナ枯渇はAhriにとって敗北の宣告だ。
2. **High Performance Combo**:
   - **Luden's Companion -> Shadowflame**
     - 現環境のバーストメタにおける絶対解。Shadowflameのクリティカル判定は、Eから始まるコンボにおいて、敵キャリーを確実に葬る「確定死」を演出する。
   - **Zhonya's Hourglass (3rd Item)**
     - 敵陣へ飛び込むための保険。敵の主要スキルを無駄遣いさせ、味方のキャリーが火力を出す時間を稼げ。

### 戦術家からの提言
Ahriは、あなたの「意図」を反映する鏡である。

敵のJinxが育つ前にゲームを終わらせるか、Zyraの序盤の勢いをロームで完全に否定するか。迷う必要はない。盤面を俯瞰し、最も脆い箇所をチャームで撃ち抜くのみ。

勝利は、計算された一歩先にある。戦場へ戻れ。
