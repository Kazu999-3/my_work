# 勝利の方程式：Pantheon vs Vi [Jungle]

チャレンジャーの頂で戦う貴公へ捧ぐ。このレポートは、一瞬の油断が命取りとなる極限環境における「勝利の設計図」である。盤面を支配し、敵を蹂躙するための絶対指針をここに記す。

### 1. 核心的メタ解釈
Pantheonの強みは「序盤の圧倒的な圧力」と「確定CCによる強引な介入」にある。対するViは、レベル6以降の無敵の突撃能力を武器とする。このマッチアップにおいて重要なのは、**「ViのパワースパイクをPantheonの先行逃げ切りで完全に無力化すること」**だ。序盤のレーン戦から主導権を握り、Viが育つ前に戦場を終わらせる。これが唯一無二の勝利条件である。

### 2. 序盤の盤面制圧
*   **ウェーブ・コントロールの徹底**: 敵のミニオンを意図的にフリーズさせ、相手を孤立させよ。敵がラストヒットを狙う一瞬の隙に、スキルと通常攻撃を叩き込む。
*   **視界の先制**: 敵ジャングラーの侵入経路をワードで封鎖せよ。相手の動線を可視化することで、自らのガンク成功率を跳ね上げ、敵のカウンターを未然に防ぐ。
*   **先読みのCC**: 敵のブリンクスキル使用を誘発させ、その硬直を狙って「引き寄せ」を叩き込む。反応ではなく、敵の行動を読み切った「先打ち」こそがチャレンジャーの証明である。

### 3. 中盤の殺し切り
*   **バックラインへの浸透**: 集団戦の開始時、敵の脆いキャリーを瞬時に識別せよ。敵の重要スキルがクールダウンに入った瞬間を狙い、バックラインへ飛び込む。
*   **AAキャンセルの徹底**: スキルとスキルの合間に必ず通常攻撃を挟め。DPS（時間当たりのダメージ）を最大化させるこのテンポこそが、敵の体力を一瞬で削り取る鍵となる。
*   **距離感の支配**: スキルがリチャージ中の間は、敵の最大射程外へ即座に退避せよ。無駄な被弾は、敵に逆転の芽を与える最大のミスである。

### 4. 【秘匿】極限ミクロ
*   **壁際スタン**: 引き寄せスキルは、敵を壁に衝突させることでスタン時間を延長させよ。これは基本だが、極限状態でもこれを外さぬ精度が勝敗を分ける。
*   **スタッターステップ**: 被弾直前の0.5秒、移動クリックを繰り返せ。射程圏内を維持しつつ、敵の主要スキルを無力化するこの歩法こそが、貴公を無敵の存在へと昇華させる。
*   **マップ視認の常態化**: 0.5秒おきにマップを確認し、他レーンの崩壊を予兆せよ。自らの戦果に溺れず、戦場全体を俯瞰し、味方の危機を救う寄りの判断を最速で行え。

### 5. 絶対回避事項
*   **過信によるダイブ**: 残り体力が僅かな状況で「フラッシュ」を温存し、デスを喫する愚行は断じて許されぬ。リスクを最小化せよ。召喚士スキルは、貴公が生き残り、敵を殺し切るためにある。
*   **視界の空白**: ワードがないブッシュには決して近づくな。敵の奇襲は「予測不可能」ではない。「予測を怠った結果」に過ぎない。

**軍師の断言**：この戦術を遂行すれば、敗北の二文字は貴公の辞書から消える。迷いを捨て、冷徹に盤面を支配せよ。勝利は必然である。以上。
