勝利の方程式

王よ、貴様がこのレポートを手に取った時点で、勝利への道は既に拓かれている。今シーズン、最速で頂点へ駆け上がるための「EASIEST Champions to CLIMB FAST」の全貌をここに記す。一切の曖昧さを排し、断定的な指示のみを貴様に与える。心して聞け。

### 1. 核心的メタ解釈

今シーズン、勝利への最短経路は、低難度で高リターンを叩き出すチャンピオンによる盤面制圧とスノーボールである。我々が選定したチャンピオンは、シンプルな操作性にもかかわらず、圧倒的なレーン支配力、尋常ならざるギャンクポテンシャル、そして集団戦における決定的な影響力を兼ね備えている。敵に考える隙すら与えず、序盤から終盤まで一方的に蹂躙する。これが貴様の目指すべき絶対的な戦術である。GarenとDariusはトップレーンを絶対的に支配し、敵を育たせない。RammusとBriarはジャングルからレーンに圧力をかけ、オブジェクトを確実に確保する。BrandとDianaは中盤以降、圧倒的なバーストと集団戦能力で試合を終わらせる。これらのチャンピオンは、貴様が持つべき「勝利の方程式」そのものだ。

### 2. 序盤の盤面制圧 (0m00s - 15m00s)

序盤は敵に一切の希望を与えない、絶対的な支配を確立する。

*   **Garen (Top):** Eスキル「ジャッジメント」でミニオンと敵チャンピオンを同時に攻撃し、レーン優先権を確保せよ。敵がトレードを試みた場合も、Eの範囲でダメージ交換を有利に進め、常に優位を維持しろ。Q「デマーシアの勇進」で突進し、Eで回転攻撃。敵のHPが30%以下であれば、R「デマーシアの正義」で即座にキルを確定させろ。W「勇気」のダメージ軽減効果はトレード開始と同時に発動し、被ダメージを抑制せよ。パッシブ「不屈」による回復を活用し、レンジチャンピオンに対してもレーンサステインを維持し、レーンに居座り続けろ。

*   **Darius (Top):** Ghostbladeラッシュで得られる移動速度を活かし、Q「皆殺しの斧」を敵チャンピオンに当てつつウェーブを処理し、レーンを押し上げろ。E「捕縛」は貴様の唯一のギャップクローザーである。これを敵に回避されることは貴様の脆弱性を露呈させる。決して外すな。Qのスタックを意識し、5スタックが溜まったらR「ノクサスのギロチン」で敵を処刑せよ。Ghostbladeラッシュにより、サモナースペルをGhostからIgniteまたはExhaustに変更し、レーンのキルプレッシャーを最大化しろ。

*   **Rammus (Jungle):** Q「パワーボール」でレーンに高速で突入し、W「守りの体勢」で敵のダメージを受けつつE「フレンジー・トーント」で敵を拘束せよ。Nunuと同様に、リソースやアイテムへの依存度が極めて低いため、序盤からギャンクを頻繁に敢行し、敵レーナーに絶え間ない圧力をかけろ。QとWのリフレクトダメージにより、序盤から高いバーストを出すことが可能だ。ADチャンピオンは貴様のサンドバッグに過ぎない。

*   **Briar (Jungle):** ギャンクは側面からのエンゲージが理想的である。基本コンボは、通常攻撃 -> W「ブラッドフィースト」 -> 通常攻撃 -> Q「突撃」 -> 通常攻撃 -> W2(第二段)。これにより、AAリセットを最大限に活用し、バーストダメージを増加させろ。Wは貴様のHPが低いほど回復量が増えるため、発動タイミングを完璧に見極めよ。Raptorsからスタートし、Red、Krugsをクリア後、Long Swordを購入し、BlueサイドをクリアするとScuttle Crabに間に合う。この完璧なジャングルルートで序盤のアイテムとレベルのアドバンテージを確保せよ。

### 3. 中盤の殺し切り (15m00s - 20m35s)

中盤は序盤の優位を決定的なものとし、試合を殺し切る。

*   **Briar (Jungle):** 敵チーム構成に応じてビルドを最適化しろ。Squishyな敵が多い場合は、早期バーストに特化したHuB Rush (Hubris -> Collector -> LDR -> IE) を選択し、試合の早期終了を狙う。耐久力が必要な場合は、BotRK -> Black Cleaver -> Death's Dance -> Guardian Angelを継続し、持続的な脅威となれ。この柔軟な対応が、勝利を確実にする。

*   **Brand (Mid/Support):** E-maxビルドに移行することで、W-Eコンボによるウェーブの即時クリアと、レーン相手へのフリーポークを恒常的に実行する。これによりレーン主導権を絶対的なものとし、敵のCS取得を阻害せよ。このウェーブ管理とポークにより、Rylai's Crystal Scepterへの到達を早めろ。Rylai's完成後は、スキルショットの精度に依存せず、敵を永続的にスロー状態に保ち、集団戦での圧倒的優位を確立する。

*   **Diana (Mid/Jungle):** レベル3まではCS重視の安全なレーニングを徹底し、QとWでヘルスを温存せよ。レベル3到達後、Qを当てた際のEダッシュ、W展開、Electrocuteプロックによる高ダメージトレードを積極的に仕掛けろ。レベル6到達後はQヒットからのR、Eリセットを駆使し、即座にソロキルを狙う。ウェーブクリアの速さを利用し、サイドレーンでの1v1優位を確立せよ。また、ローム機会を常に伺い、他レーンへの圧力として活用する。集団戦においては、耐久性とRの広範囲CCを活かし、適切なタイミングで複数の敵を巻き込み、一瞬でファイトを終わらせろ。

### 4. 【秘匿】極限ミクロ

貴様が真の覇者となるために、一般には知られていない極限のコツを伝授する。

*   **Garen:** W「勇気」のダメージ軽減効果は、敵とのトレード開始と同時に発動せよ。これにより、被ダメージを最小限に抑え、有利な交換を成立させろ。パッシブ「不屈」とQ「デマーシアの勇進」の移動速度増加を活用し、サイドレーンでは敵の捕獲を困難にさせ、あるいは敵を確実に捕らえろ。対Garenの最優先BANはKarthus, Kayleである。これらを敵に渡すな。

*   **Darius:** Ghostbladeラッシュにより、貴様の唯一の弱点であるギャップクローザーの課題を解決せよ。これによりサモナースペルをGhostからIgniteまたはExhaustに変更し、レーンのキルプレッシャーを高めるか、バーストを軽減せよ。Qを当てて回復し、有利なトレードを継続しろ。序盤のスノーボールを活かし、ミッド～レイトゲームでは防御アイテムに移行し、無敵の存在として影響力を維持せよ。

*   **Rammus:** Nunuと同様に、リソースやアイテムへの依存度が低いため、ギャンクを頻繁に敢行せよ。QとWのリフレクトダメージにより、序盤から高いバーストを出すことが可能だ。Lv6以降、R「震大地」が加わり、ギャンクはさらに強力になる。オフensiveアイテムなしでも、Q + E + WのリフレクトダメージでADチャンピオンを容易にソロキルできる。ミッド～レイトゲームでは純粋なタンクとして機能し、さらに強力になる。

*   **Briar:** W「ブラッドフィースト」はHPが低いほど回復量が増える。この特性を最大限に活かし、瀕死の状況から逆転を狙え。R「キルに飢えし暴走」が命中した場合、Wは使用せず通常攻撃に集中しろ。Raptorsからスタートし、Red、Krugsをクリア後、Long Swordを購入し、BlueサイドをクリアするとScuttle Crabに間に合う。この完璧なジャングルルートで序盤のアイテムとレベルのアドバンテージを確保せよ。

*   **Brand:** 序盤にダークシールを購入し、高いポーク能力を活かしてスタックを蓄積せよ。これにより、メジャイへのアップグレード機会を創出し、高いキャリーポテンシャルを追求する。貴様の火力は無限大となる。

*   **Diana:** ミニオンウェーブの左右端（青チームは左、赤チームは右）からQを放て。Qのカーブを利用し、ミニオンを貫通させつつチャンピオンにQを命中させる確率を最大化するのだ。これにより、ウェーブクリアとチャンピオンポークを同時に行い、レーンでの優位性を築き上げろ。

### 5. 絶対回避事項

これらの過ちは貴様の勝利を遠ざける。決して犯すな。

*   **Darius使いよ、** Eスキル「捕縛」を敵に回避されることは、貴様の命取りとなる。このスキルを外すことは、貴様が脆弱性を晒すことを意味する。決してミスを許すな。
*   **Briar使いよ、** W「ブラッドフィースト」の無謀な発動は自滅を招く。発動タイミングを完璧に見極めろ。HPが低い時、側面からの奇襲時など、状況を完璧に判断しろ。
*   **Diana使いよ、** レベル3以前の無謀なトレードは避けるべきだ。貴様のパワースパイクはそこではない。CSを優先し、レベル3、そしてレベル6の爆発的な力を待て。焦りは敗北を招く。
*   **Garenをピックする際、** KarthusやKayleをBANリストから漏らすな。これらのチャンピオンはGarenにとって絶対的な脅威となる。彼らを敵に渡すことは、貴様自身に不利な状況を強いることに他ならない。
