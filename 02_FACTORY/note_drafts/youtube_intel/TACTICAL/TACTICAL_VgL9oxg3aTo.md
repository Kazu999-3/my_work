# 勝利の方程式：Karthus vs Nidalee - KR Master Tier

KarthusでNidaleeを制圧する。それは単なる操作技術の競い合いではなく、ジャングルという盤面における「確率の支配」である。貴殿が王者として君臨するための、唯一無二の戦術をここに記す。

---

### 1. 核心的メタ解釈：リソースの非対称性
Nidaleeは序盤の機動力と爆発力で盤面を荒らす「狩人」だが、Karthusの真価は「時間経過による絶対的なスケーリング」にある。この対峙において、Nidaleeの攻勢を捌くことは最低条件に過ぎない。重要なのは、**「Nidaleeが仕掛ける場所を予測し、その裏でマップ全域の資源を奪い尽くす」**ことだ。彼女が1キルを追っている間に、貴殿は2キャンプと1タワーを削り取れ。それがKarthusの戦場支配の真髄である。

### 2. 序盤の盤面制圧：先読みの視界と制御
Nidaleeの侵入を許すな。
*   **予測的ワード配置:** 敵の現在地ではなく、敵が次に通らざるを得ない「経路」を封鎖せよ。彼女が姿を見せた瞬間には、既に貴殿の罠が完成している状態を作り出せ。
*   **ウェーブの拒絶:** 敵タワー下への無意味なプッシュは愚行だ。敵ジャングラーが不明な間は、兵士を自陣側に引き寄せ、敵を孤立させてから葬り去れ。フリーズこそが、相手の成長を止める最も残酷な兵器である。

### 3. 中盤の殺し切り：最適化されたコンボ
中盤の小競り合いで、貴殿のスキル精度は勝敗の全てを決する。
*   **脱出スキルの釣りと硬直の排除:** CCを敵の回避スキルが残っている段階で浪費するな。相手の脱出スキルを吐き出させたことを確認してから、自身のバーストを叩き込む。
*   **DPSの最大化:** スキル間に挿入すべき通常攻撃を疎かにするな。0.1秒の遅延が敵のフラッシュ逃走を許す。身体に染み付いたリズムで、アニメーションをキャンセルし、DPSを極限まで引き上げろ。

### 4. 【秘匿】極限ミクロ：王の所作
*   **カイトの極意:** アタックムーブ（Aクリック）を磨け。敵の射程圏外からクリックを開始し、敵が射程に入った瞬間に攻撃を発生させる。この「一歩の微調整」こそが、敵の攻撃を無効化し、貴殿のみが一方的にダメージを与える境界線だ。
*   **心理的誘導:** 集団戦開始前、わざと微量なダメージを受けろ。敵に「強引なイニシエート」を誘発させるのだ。敵のミスを誘うのは腕力ではない。知略による罠である。

### 5. 絶対回避事項
*   **「感覚」への依存:** 貴殿の直感は不要だ。全てを確率と計算で語れ。
*   **死角の放置:** 視界の空白は即ち死を意味する。オブジェクト争奪戦において、先読みのワード設置を怠れば、その瞬間に王座から転落する。
*   **不用意な立ち位置:** 敵のスキル射程圏内を無防備に歩くな。常に「自身の有利な距離」を維持し、敵を貴殿の射程範囲内で踊らせろ。
