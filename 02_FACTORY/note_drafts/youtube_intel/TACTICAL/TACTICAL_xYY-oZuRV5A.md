# 勝利の方程式：Cassiopeia - 盤上の毒蛇

カシオペアを操ることは、戦場に死の網を張ることに等しい。貴様がこのレポートを理解し、完璧に遂行するならば、対面する敵はただ毒に蝕まれ、抗う術を失い、崩れ落ちるのみだ。以下、勝利のための絶対規定を記す。

### 1. 核心的メタ解釈
カシオペアは単なるメイジではない。**「移動制限の支配者」**であり、敵の選択肢をゼロにする拘束装置だ。現在のメタにおいて、敵の機動力は脅威だが、それは「W（ミアズマ）」の前では無力化される。敵がブリンクを吐いた瞬間、そこは死地と化す。敵を倒すのではなく、敵の「逃げる権利」を剥奪し、逃げ場のない檻へ追い込むのが貴様の役割だ。

### 2. 序盤の盤面制圧
序盤の貴様は、狩人でありながら忍耐を強いられる。
*   **凍結の支配:** レーンを自陣タワー前でフリーズさせ、相手JGの介入を物理的に拒絶せよ。敵がロームを試みれば、その瞬間にウェーブを押し込み、タワープレートを破壊し経済的格差を確定させる。
*   **視界の構築:** 川の分岐点と敵JGのバフ位置は、貴様の庭である。コントロールワードをドラゴン前へ常駐させ、相手の視界を「盲目」にせよ。ガンクの予兆を3秒早く察知すれば、それはもはやガンクではなく、貴様にとっての「カモ」が狩場へ飛び込んできたに過ぎない。

### 3. 中盤の殺し切り
この段階で貴様は、マップ全域の支配権を握る。
*   **CCの最適化:** 味方のCCと自分のスキルを重ねる愚を犯すな。CCの持続時間を「延長」させる意識を持ち、相手を永久に拘束せよ。
*   **機動的バースト:** R（石化の抱擁）を当てる際、射程など考慮するな。フラッシュを併用したRで、敵の反応速度を上回る奇襲を仕掛けるのが基本だ。敵が石化した瞬間、Eの連打でHPバーを消滅させろ。

### 4. 【秘匿】極限ミクロ
勝敗を分かつのは、この「隠された一手」だ。
*   **Qの必中理論:** 敵のCS取得モーション、あるいはスキル硬直の瞬間にQを置け。敵がQを避ける動作を強制し、その移動先を予測してWを置く。Qは「当てるもの」ではなく「避けさせるもの」であり、避けさせた先が死地となるよう誘導せよ。
*   **リコールの美学:** 相手がアイテムを未購入かつ、主要スキルのCD中。この一瞬の隙こそが最大の好機だ。相手が反撃不能なタイミングを見極め、ウェーブを押し切ってから帰還し、常に装備差で圧倒しろ。

### 5. 絶対回避事項
これだけは肝に銘じろ。守れぬ者は敗北する。
*   **重複の禁止:** 味方のCCと自分のスキルを重ねるな。それはリソースの浪費であり、戦術的敗北だ。
*   **慢心の排除:** 敵が「見えている」時ほど、背後の闇を警戒せよ。視界内は安全ではない。視界外からの強襲こそが、貴様が唯一警戒すべき脅威だ。
*   **Qの安易な使用:** Qを外せば、貴様の殺傷能力は半分になる。焦るな。Qは確信を持って放て。Qを避ける能力こそが、カシオペア使いの真髄であると知れ。

貴様に求められるのは、慈悲ではない。冷徹なる判断と、敵の逃走を許さない無慈悲な追撃だ。戦場へ戻れ。毒蛇の牙が、敵の心臓を貫くその時まで。
