# 勝利の方程式：スモルダー・マップ境界蹂躙戦術

貴殿が今手にしているのは、スモルダーのポテンシャルを極限まで引き出し、敵を理不尽な死へと誘う「禁断の技術」である。この戦術を理解し、体現する者のみが、戦場において絶対的な優位を永続させることができる。

### 1. 核心的メタ解釈
スモルダーの本質は「スタックによる成長」ではない。**「マップ境界という聖域」を操作し、物理法則を無視した一方的な干渉を行うこと**にある。敵は画面内の情報で戦っているが、我々は画面外という次元を支配する。この非対称性が、全ての戦いにおける勝利の前提条件だ。

### 2. 序盤の盤面制圧
レーン戦において「マップ外移動」を生存のためだけに使うのは愚策である。序盤から境界線をなぞるポジショニングを徹底し、敵の射程外からスキルを叩き込め。敵がタワー下に逃げ込もうとも、境界線を利用したダイブでタワーのターゲットをリセットさせろ。敵は「なぜ攻撃が当たらないのか」と困惑する間に、スタックとキルを献上することになる。

### 3. 中盤の殺し切り
中盤の集団戦では、ウルト（AOE）を「マップ外」から放て。視覚インジケーターが敵のUIに正しく反映されないこの攻撃は、敵にとって「不可視の死」と同義だ。敵の予測ラインを無視し、死角から壊滅的な火力を叩き込む。逃げ道は存在しない。敵が生存を諦めるよりも早く、戦場を更地にするのだ。

### 4. 【秘匿】極限ミクロ
この戦術の肝は、マップの端を「滑る」ような精密なクリック操作にある。
*   **境界リセット**: タワーの攻撃弾が放たれた瞬間に境界へ逃げ込み、着弾を無効化せよ。
*   **死角投射**: 境界線ギリギリに身を置き、敵のカメラワークが追従できない角度からウルトを起動せよ。
*   **反応の超越**: 敵が「攻撃が来る」と認識した瞬間には、既に弾道は敵の懐を通過している。反応速度で勝つのではなく、認識の土俵に立たせないことが真のミクロだ。

### 5. 絶対回避事項
*   **慢心による深入り**: 境界線操作を過信し、敵のCC（移動不能スキル）を食らう位置で停止するな。それは自殺行為だ。
*   **単調なルート**: 常に同じ位置から境界へ逃げれば、敵に予測される。常に複数の逃走ルートと、境界へ向かう角度を変化させ、敵に「読み」の余地を与え続けること。
*   **中途半端な執行**: ダイブは躊躇した時点で負けだ。スキル発動のタイミングと境界への逃走を完璧に同期させよ。失敗は許されない。

**結論を述べる。**
この戦術を習得した貴殿にとって、敵のタワーも、視界の死角も、全ては勝利のための道具に過ぎない。戦場を支配せよ。慈悲は不要だ。敵が戦意を喪失するまで、この理不尽な火力を叩き込み続けろ。
