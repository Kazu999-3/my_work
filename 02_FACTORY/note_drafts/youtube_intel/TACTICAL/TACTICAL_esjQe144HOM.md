# 勝利の方程式：影の支配者ノクターン

貴殿が手にしたデータは、チャレンジャー帯の頂点に君臨するための羅針盤だ。ノクターンという駒を単なる「突撃兵」と捉えるのは三流の所業。影の支配者として戦場を支配し、敵の心に絶望を刻み込むための全戦術をここに記す。

---

### 1. 核心的メタ解釈：ノクターンは「暗殺者」ではなく「攪乱の権化」
ノクターンのRは、敵の視界を奪うことそのものが最強の武器である。敵にとっての「視界喪失」は、「味方との連携の遮断」を意味する。闇が訪れた瞬間、敵は孤立し、パニックに陥る。この心理的優位を突くことこそが、勝率を跳ね上げる鍵だ。突撃はあくまで結果であり、目的は**「敵陣の意思決定を破壊すること」**にある。

### 2. 序盤の盤面制圧：影の領域の拡大
序盤、貴殿の役割は「影の匂い」を敵全体に嗅がせることだ。
*   **フルクリアの徹底:** 6レベルまでの時間は聖域。無駄なガンクで経験値を損なうな。
*   **カウンタージャングルの圧力:** Rが上がった瞬間、敵ジャングラーは「自分が狙われている」という恐怖に支配される。この心理的圧迫だけで、敵の動きは停滞する。
*   **視界の支配:** 敵のジャングル内に深いワードを置け。闇の中で敵の位置が特定できれば、R発動後の着地点は「キル確定地点」と化す。

### 3. 中盤の殺し切り：Rの「溜め」が勝敗を分ける
Rを発動した際、即座に飛び込むのは素人の行動だ。
*   **沈黙の6秒間:** Rの視界遮断は6秒間続く。最初の2〜3秒はあえて飛ばず、敵のバックラインが孤立するか、焦ってポジションをミスするのを待て。
*   **連携の崩壊:** 敵が混乱し、サポートがADCを離したその瞬間が、狩りの開始だ。敵に「どこから来るのか？」と疑念を抱かせ、恐怖の極致で仕留めよ。

### 4. 【秘匿】極限ミクロ：ウルティ・バッファの極意
着地と同時に勝負を決める、プロトコルを叩き込め。
*   **先行入力の美学:** R飛行中にQとEを予約しろ。
*   **操作の極意:** `R(再発動) + Q + E` を一連の動作として指に覚え込ませる。着地と同時にQの攻撃力バフが乗り、Eの恐怖チェーンが繋がる。この瞬間、敵のHPバーは消滅する。このコンボに0.1秒の遅延も許されない。全神経を指先に集中させろ。

### 5. 絶対回避事項：貴殿が犯してはならない過ち
*   **「突撃の強要」を捨てろ:** 視界を奪ったからといって、必ずしも飛び込む必要はない。敵が過剰に防衛陣を敷いたなら、Rを解除し、次の機会を待て。Rを温存するだけで、敵は永遠に恐怖し続ける。
*   **孤立した標的を追うな:** 敵のメインキャリーが守られているなら、無理に突っ込むな。戦術的撤退は敗北ではない。次のRで、より確実な首を狩るための布石だ。
*   **視界の空白を恐れるな:** 敵がワードを置いていない場所こそ、貴殿の庭だ。闇を恐れるな、闇そのものになれ。
