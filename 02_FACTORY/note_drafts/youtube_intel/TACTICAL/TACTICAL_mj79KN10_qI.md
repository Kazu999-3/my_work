# 勝利の方程式：リヴァン・極限制圧戦術『Sキャンセル』

リヴァンの真価は、敵が反応を許さない速度でHPを蒸発させることにある。この戦術レポートを熟読し、貴殿の指先を神域へと昇華させよ。これより「Sキャンセル」を軸とした、対面を蹂躙するための絶対規定を布告する。

---

### 1. 核心的メタ解釈：静止による加速
リヴァンの操作において最大の敵は「硬直」である。Sキャンセルとは、コマンド入力による硬直をSキー（停止命令）で強制切断し、スキル発動の先行入力をねじ込む技術だ。これは単なる小技ではない。相手の脳が「リヴァンが動いた」と認識する前に、次のダメージ判定を叩き込むための、物理法則を捻じ曲げる戦術的優位性である。

### 2. 序盤の盤面制圧：Eの温存による支配
序盤、リヴァンのE（勇猛）は生存のための逃げ道ではない。敵の攻撃を無効化し、かつ最大火力を叩き込むための「攻撃リソース」だ。Sキャンセルを習得すれば、無理にEで距離を詰める必要はなくなる。敵のAA射程内でWやRのキャストタイムをSキーでキャンセルし、即座にQを叩き込め。Eを温存したリヴァンは、敵にとって「いつでも反撃可能かつ、追撃不可能な怪物」となる。

### 3. 中盤の殺し切り：情報の非対称性による処刑
中盤の小競り合いにおいて、敵はリヴァンのコンボ速度を予測して行動する。だが、Sキャンセルを組み込んだ貴殿のコンボは、敵の予測線を0.25秒単位で上回る。特にHydra（ハイドラ）を絡めたSキャンセルは、目視不能な速度でバーストダメージを完遂させる。敵がフラッシュを切る猶予すら与えず、画面をグレーアウトさせよ。

### 4. 【秘匿】極限ミクロ：筋肉への刻印
この技の真髄は「バッファリングの精密性」にある。
* **入力順序:** 敵のAA射程内にて、「W/R/Hydra」→「S」→「Q」を流れるような一連の動作として叩き込め。
* **Qの指定:** Qは必ず敵ユニットをターゲットせよ。地面クリックによるQは移動距離が伸び、DPSを低下させる。
* **反射神経の鍛錬:** キャストタイムが極めて短いHydraを起点にしたキャンセルは、もはや思考の領域ではない。プラクティスツールに籠もり、指が勝手に動くまで筋肉に刻み込め。これができない者は、リヴァンを使う資格がない。

### 5. 絶対回避事項：慢心と誤判断
* **過信の排除:** Sキャンセルは万能ではない。AA射程外での空振りは、最大の隙を晒すことに他ならない。常に射程を意識せよ。
* **状況判断の放棄:** W、R、Hydraのどれでキャンセルを誘発するかは、相手のクールダウンと現在の距離に依存する。定型的なコンボに固執する者は、格上の相手にその隙を突かれる。
* **練習不足の出撃:** 実戦でミスをすることは、敵に猶予を与えることと同義である。完璧な精度に到達するまで、ランク戦での試行は厳禁とする。

**軍師の総括**
リヴァンは使い手の練度がそのまま勝敗を決定づける。Sキャンセルを習得した貴殿にとって、対面の敵はもはや障害ではなく、ただの「経験値」に過ぎない。この方程式を胸に、戦場を支配せよ。勝利は、準備を怠らぬ者の手にのみある。
