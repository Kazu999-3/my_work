# 勝利の方程式：レクサイ・ジャングル戦術レポート

レクサイで高レートを維持することは、感情を排した「時間の最適化」そのものである。1100LPへの到達は、偶然の産物ではない。以下に記すのは、戦場を支配するための絶対的な指針である。

---

### 1. 核心的メタ解釈：時間の浪費は死と同義
レクサイにスケーリングの猶予はない。序盤のファームを怠ることは、自ら敗北のチケットを切る行為に等しい。
*   **フルクリアの神聖化：** 最初の2周はフルクリアを絶対とせよ。中途半端なガンクで時間を溶かす暇があるなら、キャンプを狩り尽くせ。
*   **リソース先行：** 自身のレベルが相手ジャングラーを上回っている状態こそが、レクサイの生存圏である。この優位が崩れた瞬間、貴殿の存在価値はゼロになる。

### 2. 序盤の盤面制圧：計算された侵略
盤面を支配するのは運ではなく、ドラゴンのリスポーンとキャンプのスポーンを同期させる「計算」である。
*   **ルートの固定：** サイド問わず自陣バフから開始し、全キャンプを掃討せよ。ブレッド（ラプター）をスキップするなどの小賢しい工夫は不要。効率こそが正義である。
*   **オブジェクト優先度：** 15分までにドラゴン2体の確保を完了させよ。もし失敗した場合は即座にトップサイドのタワー攻城へシフトし、ゴールドを強引に奪い取れ。

### 3. 中盤の殺し切り：効率の最大化
中盤は「不意打ち」と「タワーダイブ」の二択に集約される。
*   **ミッドの介入：** プッシュしているミッドレーナーに対するタワーダイブが、最も効率的なゴールド転換手段だ。
*   **カウンターの徹底：** 敵が動いた場合、無理に追う必要はない。相手のキャンプをすべて奪い、リソース差を広げることで、相手に「焦燥感」という名の毒を植え付けろ。0-0のスコアボードは、相手にとって最大の圧力となる。

### 4. 【秘匿】極限ミクロ：戦場の構築
トンネルは移動手段ではない。貴殿が戦場を構築するための「インフラ」である。
*   **トンネル網の構築：** 3つのトンネルを連続設置し、マップ上の機動力を極限まで高めよ。敵の視界外をなぞる「隠れ視界」を活用し、敵が予測不能な角度からノックアップを叩き込め。
*   **スマイトの保持：** 確定ダメージのリソースを無駄に掃くな。オブジェクトの確保と生存のためにのみ使用し、敵ジャングラーとの遭遇時には常に温存しておくこと。

### 5. 絶対回避事項
これらを犯す者は、いかなるレート帯であれ敗北を免れない。
*   **「無意味なガンク」の禁止：** 成功率が低いガンクは、自身の成長を停滞させるだけだ。カウンターが確定している場合のみ動け。
*   **「視界の放置」：** ピンクワードをレッドバフ付近やドラゴン前に配置し、敵の視界を物理的に遮断せよ。敵に情報を与えることは、レクサイの強みである「不意打ち」を自ら捨てることと同義である。
*   **「焦りの露呈」：** 敵の動きに振り回されるな。いかなる状況でも自身のファーム優先順位を崩さず、相手がミスを犯すその瞬間を冷徹に待て。
