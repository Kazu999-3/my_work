# 勝利の方程式：エンチャンター・セナによる戦場支配

貴様らがこれまで行ってきた「攻撃特化セナ」という愚行は、今この瞬間をもって終焉を迎える。パッチ15.21におけるセナの真価は、脆いキャリーではなく、戦場を支配する「不死の盾」としての運用にある。以下に記す戦術を完遂せよ。

### 1. 核心的メタ解釈：ユーティリティの極致
セナを単なる飛び道具と考えるのは三流の思考だ。本ビルドの核は「エコーズ・オブ・ヘリア」による回復とダメージの同時変換にある。この装備により、セナは攻撃するたびに味方を治癒し、敵を削るという二重の戦果を上げる。ダメージ交換の概念を捨て、リソースの差を圧倒的に広げる「消耗戦の支配者」こそが、このパッチにおける正解である。

### 2. 序盤の盤面制圧：エアリーの加護
ルーンに「エアリー」を刻め。これにより、攻撃と回復の両面で追加の恩恵を強制的に発生させる。
*   **至高・英気集中:** スキル回転率とステータス底上げを怠るな。
*   **生命の泉:** これをトリガーに、味方が敵を攻撃するだけで回復する環境を構築せよ。
序盤は無理にキルを狙う必要はない。射程を活かしたハラスで相手のポーションを枯渇させ、ヘリアのスタックを溜めることだけに集中しろ。

### 3. 中盤の殺し切り：ドリームメーカーによる支配
「エコーズ・オブ・ヘリア」完成後、即座に「ドリームメーカー」を組み込め。この二つのシナジーは、集団戦における味方の生存率を跳ね上げる。
*   **中盤の動き:** 敵が仕掛けてくるタイミングで、QとAAを交互に繰り出せ。敵のダメージを上回る回復量を叩き出し、敵に「削り切れない」という絶望を植え付けるのがこの戦術の真髄だ。
*   **終盤の布石:** 「ムーンストーン」で回復力を増幅させ、「ドーンコア」で全ての性能を加速させる。このビルドが完成した瞬間、敵の火力は無力化される。

### 4. 【秘匿】極限ミクロ
このビルドの真価は、ポジショニングに集約される。
*   **Qの射線管理:** 敵を攻撃しつつ、必ず味方に回復が当たる角度を維持せよ。これは「攻撃」と「支援」を同時に完遂するための必須技術だ。
*   **引き撃ちの徹底:** 敵の攻撃を受けず、かつ味方の攻撃圏内に留まる。セナの射程を最大限に活かし、敵が触れることのできない「回復の源泉」となれ。

### 5. 絶対回避事項
*   **「攻撃型アイテム」への未練:** 妖夢やコレクターを積むなどという時代遅れな選択は、即座に捨て去れ。それらは勝率を自ら下げる行為に他ならない。
*   **孤立した単独行動:** セナはサポーターである。孤立した瞬間、貴様はただの標的と化す。常に味方との距離を保ち、バフ・ヒールの連鎖を途絶えさせるな。
*   **強引なキル狙い:** 貴様の役割はキルを取ることではなく、敵に「どうやっても勝てない」という絶望感を与えることにある。

この戦術を遵守せよ。戦場の支配権は、攻撃力ではなく、確実なリソース管理を手にした者が握る。以上だ。
