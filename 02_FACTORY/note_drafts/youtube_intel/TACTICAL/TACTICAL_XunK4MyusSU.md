# 勝利の方程式：Singed vs Dr. Mundo - 対人戦術の極意

### 1. 核心的メタ解釈：リソースの優位性
ムンド戦の勝敗は「毒の効率」と「リコールの質」で決まる。Singedにとってムンドは格好の餌だが、不用意な深追いは禁物だ。ムンドのパッシブ（CC無効）を逆手に取り、毒による持続ダメージを蓄積させることに集中せよ。貴殿の役割は「倒し切ること」ではなく、「ムンドを戦闘不能な状態へ追い込み、マップ全体を貴殿の支配下に置くこと」である。

### 2. 序盤の盤面制圧：ウェーブの支配
*   **凍結の強制:** 敵ジャングラーの動向が不明な間、レーンをプッシュする行為は自殺と同義である。ウェーブをタワー手前で凍結させ、ムンドを孤立させよ。
*   **視界の深層支配:** トリンケットは通り道に置くな。敵のジャングル内、敵が必ず通る「特定ポイント」へ差し込め。情報を早期に奪うことが、無駄な撤退をゼロにする鍵だ。

### 3. 中盤の殺し切り：精密な実行
*   **オートアタックの徹底:** スキル連打に頼るな。移動スキル後の攻撃速度上昇を最大化するため、必ず通常攻撃をコンボの間にねじ込め。この0.3秒の積み重ねが、敵のフラッシュによる離脱を許さず、確実なキルを約束する。
*   **期待値の管理:** 視界なりのソロキルは低期待値のギャンブルだ。成功してもリスクに見合わない。味方の寄りと敵の配置が揃った瞬間のみ、牙を剥け。

### 4. 【秘匿】極限ミクロ：プロの呼吸
*   **入力の機械化:** 移動中、一瞬たりとも静止するな。小刻みなクリックでキャラクターを動かし続け、敵のスキルショットを無意識下で回避せよ。
*   **環境の強制最適化:** マウス加速を切り、カメラ固定を解除せよ。視界は常にマップ全体に広げ、敵のローテーションを先読みしろ。
*   **聴覚の鋭敏化:** ゲーム内BGMを切り、敵スキルの効果音を最大化せよ。視覚より先に音で敵の存在を認識するのだ。これが「反応」を「予知」に変える。

### 5. 絶対回避事項：死への誘惑
*   **「押しすぎ」の禁止:** ウェーブを押し切ることは敵ジャングラーへの招待状である。リコールタイミングはヘルスではなく、ミニオンの衝突地点と味方ジャングラーのキャンプ状況から逆算せよ。
*   **視界外への侵入:** マップの半分より先に、味方の情報なしで足を踏み入れるな。空白のエリアは、貴殿の墓場となる。
