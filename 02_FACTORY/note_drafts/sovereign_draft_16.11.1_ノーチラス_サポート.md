# 深淵のタイタン、ノーチラス：『絶対戦略』が示す、味方からの「神！」コールと勝利への支配航路

フックを外すたび、味方からの冷たい視線に怯え、サポートは地味で試合を動かせないと諦めているのか？
もう「フックが当たらないと何もできないサポート」とは呼ばせない。あなたは、試合の命運を握る『知将』へと覚醒する。

この記事は、ノーチラスをただの「フックマン」としてではなく、深淵のタイタンとして戦場を支配し、味方から「神サポート！」と称賛されるための「極秘プロトコル」だ。プロプレイヤーも密かに実践する、ノーチラス勝利戦略の全てを今、公開する。

この『絶対戦略』を掌握すれば、低レートの泥沼から抜け出し、常に戦場の主導権を握り、敵を手のひらで踊らせる『指揮官』となる快感を味わう。これは約束された未来だ。あなたはもう、ただのサポートではない。ノーチラスを解き放ち、試合の全てをコントロールする『戦場の支配者』となれ！

## 1. Core Profile
-   **Champion**: ノーチラス (Nautilus)
-   **Role**: サポート (Support)
-   **Power Spike**: Early-Mid Game (序盤の強力なエンゲージとCC、中盤のオブジェクト戦での圧倒的影響力)

## 2. Core Identity: 深淵を操る者の絶対的信条

ノーチラスの『勝利の絶対条件』は、単なる武力ではない。それは**「情報優位」**に基づいた**「決戦判断」**を核とする、圧倒的な戦術である。

「強靭な体と圧倒的なCCチェインで敵キャリーを深淵へと引きずり込み、孤立させて殲滅する。同時に、味方キャリーをあらゆる脅威から護り抜き、オブジェクトの支配権を盤石にすることで、試合を完全にコントロールする優越感を掴む。」

この信条を胸に刻み、あなたは「戦場の支配者」としての称号を手に入れるのだ。

## 3. 深淵の戦術哲学：『知略』が導く支配への道

ノーチラスの真髄は、その屈強な体と強力なCCだけにあらず。戦況を読み解き、先手を打つ「知略」こそが、あなたを勝利へと導く。

### 3.1. メタ先行のルーンとビルドパス：勝利への最適解を掴み取れ

OLEの「メタを先行せよ」の教えに基づき、ノーチラスは単一のルーンやビルドに固執しない。状況に応じた最適解を選ぶ知略が求められる。

*   **ルーン選択：状況に応じた最適な武器**
    *   **ガーディアン (Guardian)**: 基本中の基本。味方ADCの生存力を高め、安定したレーン戦を望む際に採用する。敵のバーストが激しい場合や、エンゲージにカウンターを合わせる際に特に有効だ。
    *   **アフターショック (Aftershock)**: より攻撃的なレーン展開を狙う、あるいは敵に強力な突進チャンピオンが多い場合に選択する。QやRを当てた後の耐久力を極限まで高め、キルプレッシャーを増大させる。
    *   **不死者の握撃 (Grasp of the Undying)**: 滅多に選ばれない。だが、極端なポークレーンや、自身が徹底的にタワーに張り付き、高いヘルス再生とトレード能力を求める場合に限定的に選択肢となる。
*   **アイテム選択：味方キャリーへの集中投資と柔軟性**
    *   **Knight's Vow (騎士の誓い)**: 味方ADCやメインキャリーが先行している場合、彼らの生存力を盤石にし、さらなるリードを確固たるものにする。中盤から後半にかけての集団戦で、キャリーを守り抜き「神！」と称賛されるための最優先アイテムだ。
    *   **Zeke's Convergence (ジークの収束)**: 味方ADCに高いバーストポテンシャルがある場合、エンゲージ時に更なるダメージ貢献とスロー効果を提供。集団戦であなたのイニシエートを破壊力に変える。
    *   **購入タイミングの判断基準**: 常に味方キャリーの状況と敵の脅威を評価せよ。味方ADCがデスを繰り返しているなら、まずは自身の生存力を高めるか、別のキャリーに誓いの対象を変える柔軟性を持つこと。敵にアサシンが多いなら、騎士の誓いを早めに購入し、味方を護る要塞となれ。
