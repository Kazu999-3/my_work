# Nocturne Tactical Protocol v2.0：悪夢の執行者、その絶望的支配

*「知略なき武力は、霧の中を歩くが如し。だが、我らが放つ闇は、霧そのものである。」*

味方のADCが溶け、タワーが折られ、罵声が飛び交う。そんな「負け犬の日常」を、いつまで繰り返す気だ。

LoLの戦場において、勝利とは戦うことではない。敵の意思決定を奪い、精神を破壊し、一方的に蹂躙することだ。Nocturneを操る者は、単なるジャングラーではない。敵チームに「こいつがいる限り、何もできない」と刻み込む、戦略的処刑人である。

これはメタに抗うガイドではない。メタを支配し、敵に逃げ場のない絶望を強いるための「勝利の方程式」だ。

## 1. Core Profile & Authority
- **Champion**: Nocturne
- **Role**: Jungle
- **Power Spike**: Mid Game (Lv6解禁〜2コア完成時)
- **Status**: 敵のキャリーをランク戦から「引退」させるための最強のツール。

## 2. Core Identity：闇の支配
勝利の条件はキル数などではない。敵の視界と意思決定を、R『パラノイア』で強制シャットダウンし続けることだ。
- **Rの真髄**: キルだけを狙うのは素人の所業。BaronやDragon戦、相手がオブジェクトを狩っている最中に発動しろ。それだけでSmite精度は狂い、連携は霧散する。
- **Wの重要性**: 突撃は無謀と精密の紙一重。突っ込んだ後の脆弱性は『闇の帳』で補え。主要CCを無効化した瞬間、敵は君を止める術を失い、ただ死を待つだけの存在と化す。

## 3. Timeline Strategy

### Early Game (0-15m)：「死の予兆」
- **Lv6への最短到達**: 目標は最速Lv6のみ。それ以前であっても敵のルートを追い、キャンプを奪い尽くせ。R後の着地ダメージを最大化するための金銭的優位こそが、中盤の絶対的な暴力となる。
- **孤立の誘発**: Lv6になった瞬間、マップ全体が「いつ命を奪われるか」という恐怖に支配される。これが、狙うべきプレッシャーの正体だ。

### Mid Game (15-25m)：「絶望の執行」
- **キャッチアップ**: サイドレーンで甘えた位置にいるADCやメイジは、すでに死体同然。
- **Zhonya対策**: 相手が『砂時計』を完成させたなら、それは好機。その対象を狙うふりをして他の無防備なターゲットを葬れ。敵の最強アイテムを空振らせることこそ、究極の戦術的勝利。

### Late Game (25m+)：「防衛的支配」
- **役割の転換**: 35分以降、無理な突撃は不要。味方キャリーに張り付く敵ファイターにEで恐怖を与え、ボディガードとして機能せよ。君が生きている限り、敵は不用意に陣形を崩せない。
