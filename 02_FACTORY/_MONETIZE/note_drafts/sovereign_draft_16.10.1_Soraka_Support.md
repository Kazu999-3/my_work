# 【Soraka 戦術論】星の導きが示す勝利の絶対条件 — Patch 16.10.1

*「知略なき武力は、霧の中を歩くが如し。」*

戦場を支配するのは、単なる個人の技量ではない。統計という「真実」を読み解き、時の流れを味方につけた者だけが、勝利の女神をその手中に収めることができる。

今回は、現在のメタにおける「生命の守護者」Sorakaの運用法を、戦術的視点から解剖する。

---

## 1. Core Profile
- **Champion**: Soraka
- **Role**: Support
- **Power Spike**: Mid to Late Game

## 2. Core Identity
**「味方のHPを生命線として管理し、集団戦の持続力で相手のバースト構成を無効化すること。」**
Sorakaの本質は回復ではない。相手の「キルライン」を計算外に狂わせ、敵のリソースを枯渇させることにこそ、真の価値がある。

---

## 3. Timeline Strategy

### Early Game (0-15m)
- **目標**: 視界スコア 2.0/min の確保と、Grubs戦への寄与。
- **戦術**: レーン戦では `Q (星の静寂)` を用いたハラスと自己回復で、ADCのファームを最大化せよ。敵のバーストを警戒し、ポジショニングは常に「最大射程の端」を維持すること。Grubs戦では、小競り合いが起きた瞬間に視界を確保し、中立オブジェクトへの寄与を最優先する。

### Mid Game (15-25m)
- **目標**: 1500gのリソース優位を維持するための「死なない立ち回り」。
- **戦術**: この時間帯、AatroxやZyraが仕掛けてくるMid-Gameのピークをいかに凌ぐかが鍵となる。Sorakaは集団戦の「アンカー」だ。Aatroxが突撃する際、彼にヒールを集中させ、敵のフォーカスを分散させる。Zyraの強烈なバーストには `E (星のなぎさ)` をカウンターとして使用し、敵のコンボを中断させろ。

### Late Game (25m+)
- **目標**: Jinxの「God Tier」への到達を支える究極の保護。
- **戦術**: Jinxが育ち切った瞬間、ゲームは終わる。それまでの間、君の仕事は「Jinxの生存」ただ一点に尽きる。敵の暗殺者が飛び込んでくるルートを視界で封鎖し、Jinxがパッシブを発動させるための「時間」を稼げ。君が生き残れば、勝利の確率は跳ね上がる。

---

## 4. Matchup Matrix

### Favored: Aatrox
- **理由**: Aatroxの持続的な戦闘スタイルは、Sorakaのヒールと相性が抜群だ。Aatroxが前線で耐え、Sorakaが後方から供給し続けることで、敵チームはリソース不足に陥る。

### Countered: Zyra
- **理由**: Zyraの圧倒的なゾーンコントロールとバーストは、Sorakaの立ち回りを極めて窮屈にする。植物によるハラスでHPを削り切られると、回復する暇もなく戦闘が終了してしまう。
- **対策**: Zyraのスキルショットを回避しつつ、彼女のULT発動時に `E` を重ねて無力化する準備を怠るな。無理にトレードせず、ADCと連携して視界外からの奇襲を待つのが賢明だ。

## 戦術家の提言
メタは循環する。Grubsの優先度が高い今、Sorakaは「守り」のチャンピオンではなく、**「持続的なオブジェクト確保を可能にする戦術的基盤」**として機能させよ。

視界スコア 2.0/min。この数字は単なる目標ではない。敵の動きを可視化し、君のヒールが届く範囲を地図にするための「最低限の投資」だ。

戦場へ戻れ。星の光は、勝利を信じる者のためにある。
