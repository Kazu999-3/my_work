# 勝利の運命を掴み取れ！Twisted Fate 戦術論 (Patch 16.10.1)

## 0. パッチ16.10.1のメタ洞察
パッチ16.10.1におけるTFの強みは、グローバルに影響を与えられるメイジが少なく、かつバーストダメージとピックアップが重視されるメタにおいて、彼の『デスティニー』と『ゴールドカード』が完璧に合致する点にある。特定のアイテム調整やチャンピオンのバフ・ナーフが、TFのパワースパイクを生み出し、有利なマッチアップをもたらす。この好機を逃すな。

## 1. Core Profile
-   Champion: Twisted Fate (TF)
-   Role: Mid
-   Power Spike: Mid Game (2コア、特にリッチベイン・ゾーニャ完成時)

## 2. Core Identity
このチャンピオンにおける「勝利の絶対条件」を1文で定義する。
「『デスティニー』によるグローバルな介入と、的確な『ゴールドカード』でのピックアップが、チームのオブジェクト獲得とリソース優位を決定づけ、諸君を勝利の立役者へと導く。」

## 3. 運命を操る選択：ルーンとサモナースペル
君のプレイスタイルとパワースパイクを決定づける、運命の選択だ。

### ルーン
主に『電撃』をキーストーンとし、突発的なバーストダメージとピックアップ能力を最大化する。電撃を起点としたQ+AAコンボで、貴様は敵を瞬時に葬り去る力を得る。
サブツリーには『天啓』を選び、ファーム支援と中盤のアイテム加速を図る。『パーフェクトタイミング』でゾーニャの完成を早め、『ミニオンヘルススチール』で確実なCSを確保せべし。
特定の相手（耐久力の高いタンクや複数アサシン構成）に対しては『フェイズラッシュ』も選択肢に入るが、その場合はビルドやプレイスタイルも調整が必要となる。敵のCCが非常に重い場合は『不滅』の『ボーンアーマー』や『不滅の心』も考慮に入れ、生存性を高めよ。

### サモナースペル
基本は『フラッシュ』と『イグナイト』でレーンキルプレッシャーとピックアップ力を高める。イグナイトはゴールドカードとの組み合わせで、敵キャリーに逃げる隙を与えない。
敵のCCが重い場合は『クレンズ』で反撃の機会を伺え。スプリットプッシャーへの対応や、ULTのクールダウン中にロームが必要な場合は『テレポート』も考慮に入れよ。これらの選択は、マッチアップとチーム構成によって柔軟に変えるべきだ。もはや運任せではない。貴様が状況を判断し、最善の選択をするのだ。

## 4. 支配への道筋：アイテムビルドパス
コアアイテムの順序、初手、そしてブーツの選択は、貴様の成長曲線を決定づける。

### 初手
『ドランリング』とポーションでレーン維持力とファーム効率を確保せよ。これにより、序盤のゴールド差を確実に築く。

### ブーツ
『ソーサラーシューズ』で魔法貫通を確保しダメージを増強するのが基本。敵を蹂躙するための必須アイテムとなる。しかし、敵のCCが重い、またはADダメージが深刻な場合は『マーキュリーブーツ』や『プレートスチールキャップ』も選択肢。生存性を確保し、反撃の機会を待て。

### コアアイテムの優先順位
1.  **最初のバック**: 『ロストチャプター』を最優先し、マナの問題を解決、Qによるウェーブクリアを強化する。あるいは、敵アサシンが多い場合は『ストップウォッチ』を優先する大胆な判断も求められる。
2.  **コア1**: 『エバーフロスト』または『ルーデンエコー』でマナとバーストを確保せよ。エバーフロストは追加のCCと生存性を、ルーデンエコーは爆発的なバーストダメージを提供する。メタやマッチアップに応じて、どちらを選ぶか見極めよ。
3.  **コア2**: 状況に応じて『リッチベイン』でAA強化バーストを、または敵の脅威に応じた『ゾーニャの砂時計』を完成させる。アサシンが猛威を振るうメタであれば、初手でストップウォッチからゾーニャを急ぐ判断も極めて重要だ。ゾーニャは、ULTで突っ込んだ際の生存性を高め、敵の反撃を無力化する。リッチベインは、AAのバーストを圧倒的なものとし、貴様のキルポテンシャルを飛躍的に向上させる。
4.  **4手目以降**: 『ラバドンデスキャップ』でダメージを最大化するか、『ヴォイドスタッフ』で魔法防御貫通を確保する。敵の防御を粉砕し、勝利を決定づけろ。
