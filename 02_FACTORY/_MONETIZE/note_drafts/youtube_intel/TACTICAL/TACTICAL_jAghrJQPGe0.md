# 勝利の方程式：2026年版・絶対無敵の岩山（Malphite）運用術

貴殿が求めるのは勝利の再現性だ。2026年のメタにおいて、Malphiteは単なるタンクではない。戦場を支配し、敵の心を折る「理不尽の体現者」である。以下の戦術を脳に刻み込め。迷いは敗北に直結する。

---

### 1. 核心的メタ解釈：岩の意志
この戦術の根幹は「不動」にある。Malphiteの強みは、敵のAD攻撃を無効化する圧倒的な物理防御と、一撃で戦況を覆す強力なCCだ。2026年の環境では、ただ耐えるのではなく「敵のテンポを強制停止させる」ことが求められる。相手がどれほどテクニカルに動こうとも、岩の前では無力であるという事実を突きつけろ。

### 2. 序盤の盤面制圧：リソースの枯渇
*   **パッシブの絶対管理:** 「花崗岩の盾」は敵の攻撃タイミングを狂わせるための盾だ。常にシールドが回復する間隔を意識し、無駄な被弾を避けてトレードの主導権を握れ。
*   **Qによる拒絶:** 敵がウェーブに触れようとした瞬間、Qを叩き込め。減速とノックバックで敵の動きを阻害し、前衛ミニオンへの攻撃を妨害せよ。
*   **「凍てつく心」の早期着手:** AD対面であれば、迷わず「凍てつく心」を完成させろ。相手の攻撃速度を奪うことで、レーン戦における対面の機能は完全に停止する。

### 3. 中盤の殺し切り：不可避の連鎖
*   **リセットの芸術:** HPが閾値を下回るか、キーアイテム購入の好機が訪れたら、躊躇なくウェーブを押し込みリセットを強要せよ。この隙に帰還し、装備差を拡大させる。
*   **CC連携の最適化:** 敵のスキル使用後のクールダウンを逃すな。QとE（地殻変動）を組み合わせ、移動速度を奪った後に追撃を叩き込む。全スキルを一度に吐き出すのは愚策だ。温存と発動のメリハリが、敵を逃さない執念を生む。
*   **視界の罠:** ブッシュを最大限に活用し、自らの位置情報を隠蔽せよ。敵が「どこにいるか分からない」という恐怖こそが、最大の防御であり攻撃である。

### 4. 【秘匿】極限ミクロ：岩山の真髄
*   **ダメージ計算の解像度:** 敵のHPバーを見て、自らのフルコンボで「殺し切れるライン」をミリ単位で把握せよ。計算外の動きは不要。確実なキルのみが盤面を支配する。
*   **無駄なAAの排除:** 不要なオートアタックは隙を生むだけだ。スキル射程を維持し、常に敵の攻撃をミニオンと自身のノックバックで相殺し続けろ。
*   **死なないことへの執着:** 突出したキルを狙う必要はない。経験値を吸い、敵の攻めを封じ、ただ「そこに立ち続ける」だけで敵のリソースは枯渇する。敵が焦って仕掛けた瞬間こそ、勝利が確定する時だ。

### 5. 絶対回避事項：敗北への扉
*   **過剰な追撃:** キルを急ぐあまり、敵のジャングラーの視界外へ深追いするな。自軍の視界がない場所への踏み込みは、死と同義である。
*   **リソースの無駄遣い:** マナとクールダウンを無計画に消費するな。敵が仕掛けてくるタイミングで合わせるため、常にQとWを温存しておく余裕を持て。
*   **タワーダイブの独断:** 自身のHPが万全でない限り、ターゲットを奪うな。ミニオンを先行させ、防御リソースを温存する鉄則を忘れるな。
