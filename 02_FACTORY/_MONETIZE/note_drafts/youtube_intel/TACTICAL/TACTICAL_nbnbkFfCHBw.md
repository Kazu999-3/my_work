# 勝利の方程式：MALPHITE - 盤面支配の極意

岩の如き不動の意志と、山を崩す破壊の衝動。マルファイトという兵器を使いこなすための戦術レポートをここに記す。迷いは死、躊躇は敗北だ。この理に従え。

### 1. 核心的メタ解釈
マルファイトの真髄は「Qの鈍足による起点作り」ではない。「敵の計算を狂わせるタワーダイブの圧力」にある。敵はタワー下を聖域と信じているが、貴殿にとってそこは単なる「処刑場」に過ぎない。この認識の差が、序盤のレーン戦における絶対的な格差を生む。

### 2. 序盤の盤面制圧
ウェーブ管理は「タワーの攻撃対象」を自在に操るためのチェス盤である。
*   **ミニオンの同期：** 味方ミニオンをタワー内に押し込む際、敵がタワーの防壁を頼みにしている瞬間こそが好機。
*   **アグロの制御：** 敵をタワー射程内に留めつつ、自身はタワーの攻撃範囲外からQで削る。敵が恐怖で引いた瞬間、退路を断つための「斜めフラッシュ」を準備せよ。

### 3. 中盤の殺し切り
キルを奪う際、焦りは最大の敵だ。
*   **コンボの最適化：** Q（加速/鈍足）→フラッシュ（位置調整）→E（攻撃速度低下/追撃）→AA。
*   **タワー・アグロの管理：** 敵を仕留めきる直前、タワーのターゲットを吸い込むのは三流だ。AAのタイミングは敵のHPが「一撃で消し飛ぶ」圏内まで徹底的に待て。焦ってAAを叩き込むな。スキルCDの回転とタワーの攻撃間隔を脳内で同期させろ。

### 4. 【秘匿】極限ミクロ
この戦術における最大の秘匿事項は**「バックステップ・ウェイト」**である。
*   **0:13の教訓：** 攻撃が早まりタワーの標的になった場合、即座に攻撃を中断し、タワーの射程外へ一歩引く。この「0.5秒の沈黙」が、タワーのターゲットをミニオンへと戻し、貴殿の生存と確実なキルを両立させる。キルを欲して被弾するな。キルは「奪う」ものではなく、盤面の結果として「付いてくる」ものだ。

### 5. 絶対回避事項
*   **先行AAの禁止：** 敵が射程外へ逃げ出せる隙を与えるAAの先行入力は、ダイブ成功率を著しく低下させる。必ずスキル（Q/E）で鈍足を付与してからAAを叩き込め。
*   **フラッシュの無駄打ち：** 敵の逃走先を予測してのフラッシュは二流。敵の退路を封じ、かつタワーの攻撃範囲を強引にコントロールするための「位置取り」としてのみフラッシュを使え。

岩は砕けぬ。貴殿の操作が正確である限り、敵のタワーは貴殿の味方となる。次なる戦場でも、この理を証明してみせよ。以上だ。
