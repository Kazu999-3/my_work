# 勝利の方程式：タリック・オペレーション戦術詳報

戦場を支配する「動く拠点」、タリックの運用において迷いは敗北を意味する。解析された全データを統合し、勝利を確定させるための絶対的指針をここに提示する。

### 1. 核心的メタ解釈
タリックは単なる回復役ではない。**「スキル回転によるパッシブ（AA）の連鎖」こそが本体である。** 
ルーンは相手のバースト耐性とスケーリングを天秤にかけ、ガーディアン（安定）かアフターショック（対オールイン）を選択せよ。いずれの構成であれ、「リバイタライズ」は必須である。マナ管理の要として「冷静沈着」を積み、リソースの枯渇を物理的に排除せよ。

### 2. 序盤の盤面制圧
*   **Wiggle-Wiggleの徹底:** Eの予備動作中、棒立ちは死を招く。左右への微細なステップで敵の予測を撹乱し、射程と立ち位置を制御せよ。
*   **パッシブの黄金律:** 「E→AA→AA→W→AA→AA→Q→AA→AA」のループを体に刻み込め。スキル使用後の2回の通常攻撃によるCD短縮を怠る者は、タリックを扱う資格がない。
*   **プッシュの判断:** 相手の機動力が低い場合、序盤から主導権を握りタワーを削る。視界がない場所からのEは、敵を攻撃範囲ギリギリに誘い出す「餌」として機能させよ。

### 3. 中盤の殺し切り
*   **ナッシャーの覇道:** 現環境においてナッシャーは必須のコアアイテムである。CDRと攻撃速度の向上は、スキルの回転率を劇的に高め、タリックを単独でもキルを狙える「戦闘兵器」へと変貌させる。
*   **20分地点の極意:** ナッシャー完成後、味方と共にオブジェクトを強襲せよ。敵の視界外から接近し、スタンからのバーストで一人を即座に排除。数的優位を盤石にすることが、勝利への最短ルートだ。

### 4. 【秘匿】極限ミクロ
*   **R（宇宙の輝き）の先出し:** 敵のバーストを見てからでは遅い。敵のイニシエート直後、味方が被弾する「直前」に詠唱を開始せよ。2.5秒のディレイを計算し、敵のCCチェインを無敵時間で無効化する。
*   **W-E-Eの連鎖:** W（シールド）を展開した瞬間にEを始動せよ。シールド爆発のタイミングとスタンの着弾を重ねることで、敵の回避を物理的に不可能にする。
*   **デスを恐れるな:** 自身がデスしても、それがチームのエースを守り切り、敵を壊滅させるための「犠牲」ならば、それは軍師の勝利である。

### 5. 絶対回避事項
*   **孤立の禁止:** 視界確保のために単独で動くことは自殺行為である。視界は常に味方（特にジャングラー）との連携で構築せよ。
*   **マナの無駄撃ち:** プレセンス・オブ・マインドの恩恵を受けられない状況でのE連打は、死を招くマナ枯渇を招く。
*   **静止の禁止:** 戦闘中、移動を止めることは死を意味する。常にマウスカーソルを動かし、敵のスキルショットを回避せよ。**「静止は弱さである」。**

以上、この戦術を完遂せよ。計算通りの勝利は、貴殿の操作の先にある。実行あるのみ。
