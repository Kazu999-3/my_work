# 勝利の方程式：Season 16 ジャングル支配術

王よ、S16の戦場を支配するための「絶対の理」をここに記す。もはや迷いは不要だ。この指針に従う者のみが、召喚師の谷における唯一の勝者として君臨する。

### 1. 核心的メタ解釈：効率という名の暴力
S16において、ジャングルは「狩猟場」から「経済圏」へと変貌した。クリア速度の低下は、裏を返せば「ルート最適化が勝敗を分かつ」ことを意味する。フルクリアの徹底こそが、レベル6への最短距離であり、中盤以降の圧倒的格差を生む源泉だ。闇雲なガンクで経験値を捨てるな。効率こそが、勝利を担保する最強の武器である。

### 2. 序盤の盤面制圧：計算された先制攻撃
ヘルス維持能力の向上により、序盤のダイブは以前よりも遥かに安全かつ強力な選択肢となった。
*   **ルート:** 2回フルクリア＋カニ1体。これで確実にレベル6へ到達し、パワーの差で盤面を制圧せよ。
*   **ダイブ:** 敵のヘルスが削れた瞬間、躊躇なくタワー下へ飛び込め。S16のジャングルは、生存能力を味方につけた者が勝つ。

### 3. 中盤の殺し切り：集団の力学
オブジェクト攻略において、ソロでの強行は愚策である。敵のタワーや中立オブジェクトは、周囲の味方人数に応じて防御ステータスが変動する。
*   **戦術:** 味方を必ず随伴させろ。味方の人数を揃えることで、敵の防御バフを逆手に取り、最短でオブジェクトを粉砕する。孤軍奮闘は敗北の入り口だ。常に「集団での破壊」を意識せよ。

### 4. 【秘匿】極限ミクロ：視界と速度の魔術
勝敗を分かつのは、地図上の「見えない支配」である。
*   **フェイライトの罠:** 敵に見えないフェイライト上で待機せよ。相手が視界内だと信じ込んでいる場所こそが、最大の死角となる。スイーパーによる無効化を駆使し、相手の「安全」を「恐怖」へと塗り替えろ。
*   **高速機動の極意:** 35キャンプ処理で得る移動速度バフは、戦場の支配権そのものだ。非戦闘時は最短距離でサイドレーンをカバーし、戦闘開始直後にバフが半減することを見越して、あらかじめ敵の退路を断つポジショニングを取れ。

### 5. 絶対回避事項：敗北への扉
*   **無策なガンク:** 経験値効率を無視したガンクは、自ら敗北のチケットを切る行為に等しい。
*   **孤立:** どんなに育っていようと、オブジェクト攻略時に味方を放置するな。敵の防御ステータス上昇を許すことは、時間を浪費し、逆転の隙を与えるだけだ。
*   **甘え:** 戦闘開始時の移動速度バフ半減を計算に入れず、敵の懐に飛び込みすぎるな。バフが消えた後、どうやって戦場を離脱するか、あるいは殺し切るか。その後のプランがない突撃は、ただの自殺である。

王よ、盤面を支配せよ。効率を極め、視界を欺き、集団で蹂躙せよ。勝利は、冷徹な計算の上にのみ存在する。
