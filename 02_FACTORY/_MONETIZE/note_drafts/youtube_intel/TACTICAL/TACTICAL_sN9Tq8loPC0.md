# 勝利の方程式：Hwei 運用戦術レポート

### 1. 核心的メタ解釈：Hweiは「砲台」にあらず「罠師」である
多くの者はHweiを広範囲魔術師と誤認し、スキルを乱射してマナと位置を浪費する。愚か極まりない。Hweiの本質は**「敵の挙動を誘導し、カウンターで粉砕する罠師」**である。Eスキルを攻撃に組み込む思考を捨てろ。Eは敵の突撃を拒絶するための「聖域」であると定義せよ。

### 2. 序盤の盤面制圧：QWによる持続的支配
レーン戦においてEを封印せよ。敵が畏怖すべきは、常にEを保持しているという「圧力」そのものだ。
*   **ハラスの鉄則:** QとWのみで構成されるコンボで、敵のHPを削り続けろ。
*   **盤面の支配:** 敵がCSを取る瞬間、あるいは射程に入った瞬間にQQ/QEを叩き込み、WQで即座に位置を調整せよ。Eを隠し持つことで、敵は「いつCCが飛んでくるか」という疑心暗鬼に陥り、CSを捨てるか、強引に突っ込んで自滅するかの二択を強いられる。

### 3. 中盤の殺し切り：リアクションによる完封
中盤の小規模戦において、Hweiの役割は先制攻撃ではなく「後出しによる制圧」だ。
*   **キルラインの設定:** 敵のダッシュスキルやブリンクスキルを誘い込め。敵が移動技を使用した瞬間、その着地点に向けてEQを置け。
*   **確定死の演出:** 敵の移動硬直にEQを重ねれば、回避は不可能だ。その後、Rを重ねて逃げ道を塞ぎ、QWで追撃を完遂せよ。予測で撃つな。敵の行動を「見てから」叩く。このリアクションの精度が、戦場の支配権を決定づける。

### 4. 【秘匿】極限ミクロ：Eの「保留」がもたらす無敵時間
Hweiの生存率は「Eを使わない時間」に正比例する。
*   **極限の自制:** 敵がジャングラーを呼び寄せても、Eを温存し続けろ。敵はEが吐かれるまでフルバーストを躊躇する。この「心理的膠着」こそが、Hweiが最も輝く瞬間だ。
*   **奥義:** 敵の飛び込みに対し、EQを足元に置くのではない。**「敵の移動先」にEQを置く。** 敵が突進した瞬間にCCが発動するよう、視覚情報の0.1秒先を支配せよ。Eは攻撃スキルではなく、敵の行動を強制終了させる「強制停止スイッチ」であると心得よ。

### 5. 絶対回避事項：慢心による「スキル全吐き」
以下の行為は敗北への直行便である。即刻禁じろ。
*   **Eをハラスに使用すること:** Eを外した瞬間、貴殿はただの的になる。敵の突撃を防ぐ術を失ったHweiに存在価値はない。
*   **予測撃ちによる無駄打ち:** 敵の動きを予測してスキルを撃つな。Hweiのスキルは「敵が逃げられない場所に置く」ものだ。敵の行動を固定してから攻撃を開始せよ。
*   **位置取りの固定:** 常にWで移動速度を確保し、常にEを構えろ。この緊張感を維持できない者は、Hweiを握る資格がない。

Eを隠し、QWで蹂躙し、敵の突撃をEQで粉砕せよ。
