# 勝利の方程式：Evelynn - 影の処刑人

戦場を支配せよ。イブリンを操ることは、単なる暗殺ではない。敵の精神を破壊し、マップ上の「存在しないはずの恐怖」を具現化する作業だ。以下の戦術を遵守し、敵陣営を絶望の淵へ叩き込め。

---

### 1. 核心的メタ解釈
イブリンの真髄は「キル」ではなく「恐怖による行動制限」にある。敵が姿を見せないお前を恐れ、不用意な動きを封じた瞬間、既に勝負は決している。お前はただのジャングラーではない。敵のバックラインにとっての「恒常的な死の予感」であれ。

### 2. 序盤の盤面制圧
Lv6までは忍耐の刻だ。無理なガンクは不要。迅速にLv6へ到達し、パッシブの隠密能力を解禁せよ。
*   **効率重視の周回:** スキル回転率を最大化し、移動時間を極限まで削減せよ。
*   **Wの心理戦:** 敵のレーンに対し、あえてWのチャームを付与して圧をかけろ。敵が下がれば味方は有利になる。無理に仕掛ける必要はない。敵の精神を削るだけで、レーン戦は勝利に傾く。

### 3. 中盤の殺し切り
Lv6以降、お前はマップ上のどこにでも存在し得る。
*   **コンボの定石:** `W → Q → E → Q → AA → Q → R`。この一連の動作を筋肉に刻み込め。
*   **バーストの最大化:** Eの強化状態を常に意識しろ。敵の逃走を許すな。Eをミニオンに叩き込み、移動速度ボーナスを得て距離を詰める「加速アプローチ」は必須テクニックだ。
*   **ターゲット選定:** 敵のキャリーを孤立させろ。Wでマークし、敵がパニックを起こして無駄なスキルを吐いた瞬間、その隙を縫って処刑せよ。

### 4. 【秘匿】極限ミクロ
凡百のイブリン使いと差をつけるための極秘事項だ。
*   **反応速度の超越:** `Flash → E` の入力を完璧にせよ。Eの射程内でフラッシュを挟むことで、敵が反応する時間をゼロにする。
*   **AAキャンセル:** Eの発動直後に移動入力を入れることで、硬直を消去し、AAを即座に叩き込め。このコンマ数秒の差が、敵のフラッシュ逃走を許すか、確実に仕留めるかの分水嶺となる。
*   **Rによる離脱と追撃:** Rは火力源であると同時に、最強の離脱手段だ。タワーダイブ後、壁を越えて無傷で帰還するまでが仕事である。

### 5. 絶対回避事項
これらを犯す者は、勝利を自ら放棄したと見なす。
*   **視界への無警戒:** 敵のコントロールワード、タワーの射程、そして味方の視界が消えた場所。これらを軽視して透明化を過信するな。敵に検知された時点で、イブリンの優位性は消失する。
*   **無謀な先出し:** 敵のCCスキルが残っている状態で突入するのは自殺行為だ。敵が主要スキル（スタンや沈黙）を吐くまで、影の中で待機せよ。焦りは死を招く。
*   **リソース管理の怠慢:** `Flash + R` や `Flash + Smite` の連携ミスは敗北に直結する。キルラインを常に計算し、確実な確信を持ってトリガーを引け。

**結論:** お前が影に潜む限り、敵に休息はない。迷わず、慈悲なく、敵のバックラインを抹殺せよ。それが勝利への唯一の道だ。
