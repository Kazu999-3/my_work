# 勝利の方程式：Keria流・Luxの支配的戦術

戦場を掌握する者のみが勝利を冠する。Keriaが体現したLuxの序盤戦術は、単なるハラスではない。相手の精神を削り、リソースを枯渇させ、戦場の物理的支配権を奪う「完全包囲網」である。以下にその極意を記す。

### 1. 核心的メタ解釈
Luxの本質は「射程の暴力」にある。相手がCS（ミニオン）を触るという最も無防備な瞬間に、こちらの攻撃を強要する。これは単なるダメージ交換ではない。相手に「CSを取れば殴られる」という恐怖を植え付け、レーン戦の主導権を物理的接触以前に掌握する心理戦である。

### 2. 序盤の盤面制圧
Lv1から勝負は決している。ミニオンを自陣の安全圏へ引き寄せることで、相手を前進させるリスクを最大化させろ。敵がCSを取得するその刹那、Qを射出する。避けられればそれで良い。重要なのは、相手が回避行動を取ることでCSを落とすことだ。この「CSかヘルスか」の二択を強要し続けることで、ウェーブ管理の主導権を完全に掌握する。

### 3. 中盤の殺し切り
ヘルス・アドバンテージを確立した後は、徹底的にリソースの差を広げろ。相手がリコールを余儀なくされるまで、決して追撃の手を緩めるな。Manaを温存しつつ、AA（通常攻撃）によるパッシブ発動を軸に、スキルを「トドメの切り札」として温存せよ。相手がリコールした瞬間、ウェーブをタワーに押し付け、Lv差とアイテム差を決定づける。これが我々の勝利への最短ルートだ。

### 4. 【秘匿】極限ミクロ
*   **境界線の支配：** 立ち位置は常に「相手の最大射程の1歩外」。相手がスキルを撃とうと踏み出した瞬間、こちらの射程内に入るよう微調整を行え。この「0.1歩の差」が、被弾をゼロにし、こちらの攻撃を100％通す絶対領域となる。
*   **AAの挟み込み：** スキル間に必ずAAを差し込め。Luxのパッシブを誘発させない攻撃は、ただの弾切れだ。常に「スキル＋AA」をセットとして思考せよ。

### 5. 絶対回避事項
*   **無意味なQの浪費：** 確信のないQを撃つな。Qがクールダウン中のLuxは、ただの的である。相手に「Qを撃たせたい」と思わせ、あえて撃たせずにプレッシャーをかけ続けろ。
*   **位置の固定：** 常に動け。立ち止まることは死を意味する。相手のスキルショットを誘いつつ、常に予測不能な軌道で動くこと。

以上がKeriaの戦術を解体し、再構築した勝利の鉄則である。この理を遵守せよ。戦場において慈悲は不要、あるのは計算された破壊のみだ。
