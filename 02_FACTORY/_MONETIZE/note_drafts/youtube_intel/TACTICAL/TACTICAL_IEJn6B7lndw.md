# 勝利の方程式：Nocturne Jungle 支配戦術

ノクターンは単なる暗殺者ではない。戦場を支配し、敵の精神を摩耗させ、必然的な敗北を突きつける「断罪者」である。以下の戦術を脳髄に刻み込め。

### 1. 核心的メタ解釈
ノクターンの真価は「キル」そのものにはない。**「敵の行動圏を制限する不可視の圧力」**こそが本質だ。
敵JGがマップに現れない時、それはノクターンがそこにいる可能性を意味する。常に視界外に身を置き、敵レーナーに「いつ飛んでくるか分からない」という強迫観念を植え付けろ。敵が守備的になれば、それだけで味方レーンは勝手に優位を築く。お前はフィールドに存在するだけで、既に勝負を決定づけている。

### 2. 序盤の盤面制圧
*   **「視差」による介入:** 敵JGの動向を予測し、相手が視界を求めて動く直前にウェーブ管理を完了させろ。キル後のウェーブ処理は、敵の増援を誘い出すための餌だ。
*   **犠牲の最適化:** 味方の生存が最優先だ。自身のキャンプを捨ててでも、敵JGのルートを阻害し、敵の選択肢を奪え。盤面の主導権を渡すな。

### 3. 中盤の殺し切り
*   **スキル運用の絶対則:**
    *   **Q:** 敵の退路を予測し、移動速度バフを維持したまま追い詰めろ。Qのバフがないノクターンはただの肉塊だ。
    *   **E:** 敵のフラッシュを誘い出すための「鎖」だ。リンクが切れない距離を維持し、敵が逃げ場を失った瞬間に叩き落とせ。
*   **Ultの神髄:** Ultは「キル発生器」ではなく「数的不利の強制生成機」だ。孤立した敵を狩るか、集団戦で敵のキャリーを孤立させるために使え。また、バロンを「敵を誘い出すための撒き餌」として活用し、寄ってきた敵の陣形が崩れた瞬間を狩り取れ。

### 4. 【秘匿】極限ミクロ
*   **先出しのQ:** 敵のダッシュやフラッシュを予測し、敵の「着地点」にQを置け。反応して避けるのではない、お前のQに敵が飛び込んでくる状況を作るのだ。
*   **pingによる精神撹乱:** 敵JGの位置を把握した際、即座にpingで味方に共有しろ。これは単なる情報伝達ではない。敵の攻撃的な動きを鈍らせ、自滅を誘う心理戦だ。敵の思考を支配せよ。

### 5. 絶対回避事項
*   **Ult無き小競り合い:** Ultがない状態での戦闘は、ノクターンにとって最大の脆弱性だ。数分間の沈黙を守り、ファームと視界確保に専念しろ。焦りは死を招く。
*   **過剰追撃の悪癖:** キルは「差を広げる手段」に過ぎない。タワー、ドラゴン、ヘラルド。これら目標の破壊こそが至上の目的だ。欲に駆られて深追いし、デスを献上するのは三流の所業と心得よ。

**戦場に感情を持ち込むな。この戦術を機械的に遂行せよ。勝利は必然だ。**
