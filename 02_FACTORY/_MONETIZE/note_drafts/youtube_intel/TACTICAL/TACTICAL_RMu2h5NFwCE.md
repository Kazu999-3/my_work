# 勝利の方程式：Singed Carry vs Jhin - KR Challenger Patch 26.03

### 1. 核心的メタ解釈：リソースの非対称性
Jhinとの戦いは「距離の管理」と「リソースの枯渇」に集約される。奴の4発目の弾丸と罠を無効化し、貴殿の毒（Q）による持続的なエリア制圧で奴の行動範囲を極限まで狭めろ。**Jhinの強みは「瞬間的なバースト」だが、貴殿の強みは「盤面の持続的支配」だ。** 奴を追い詰めるのではなく、奴が「戦わざるを得ない状況」を強制的に作り出すことこそが、勝利の唯一の道である。

### 2. 序盤の盤面制圧：兵士の支配と視界の奪取
*   **ウェーブ・コントロール:** 敵のフリーズを許すな。ミニオンのHPを微調整し、タワー射程ギリギリで相手に処理を強要させろ。貴殿が押し付けるのではなく、相手に「押し出させる」状況こそが、Jhinを孤立させる最適解だ。
*   **情報の奪取:** ワードは「守り」ではない。「敵の未来を予測する鍵」だ。敵ジャングラーのキャンプサイクルを逆算し、奴が次に現れるルートに視界を置け。敵の影が見えていない場所への突入は、ただの自殺行為に過ぎない。

### 3. 中盤の殺し切り：隙の徹底排除
*   **AAの介入:** 貴殿の敗因はコンボの甘さにある。スキルを当てることだけに執着するな。スキルとスキルの間に、必ず通常攻撃（AA）を差し込め。この0.5秒の積み重ねが、Jhinのような脆弱な標的を確実に葬るキルラインを形成する。
*   **クールダウンの狩り:** Jhinがスキルを吐いた瞬間、そこが奴の墓場だ。感情的な追いかけは不要。奴の「弾数」と「クールダウン」を数え、最も無防備な瞬間にのみ死を叩き込め。

### 4. 【秘匿】極限ミクロ：王者の領域
*   **視線の周波数:** 3秒に一度はミニマップを見ろ。敵の姿が映る前に、敵の「不在」から奴らの意図を読み取るのだ。これができれば、ギャンブル的なダイブは不要になる。
*   **逃走の計算:** 敵をタワー下に追い込み、ゴールドと経験値を奪うだけで貴殿の勝利は確定する。無理にキルを狙わず、デスを避けて経済格差を広げろ。生き残ることこそが、相手にとって最大の絶望となる。

### 5. 絶対回避事項：敗北への扉
*   **無策の突撃:** 視界なき場所への深追いは一切禁止する。
*   **リソースの浪費:** 敵のブリンクやフラッシュを確認する前に、全てのスキルを吐き出すな。貴殿のスキルは「ダメージ源」ではなく、相手を逃がさないための「制圧の檻」であると再定義せよ。
*   **慢心:** 盤面のゴールド差を忘れるな。個人の力で勝てるのは低レートまでだ。チャレンジャーの戦場では、経済差とリソースの管理こそが真の力である。
