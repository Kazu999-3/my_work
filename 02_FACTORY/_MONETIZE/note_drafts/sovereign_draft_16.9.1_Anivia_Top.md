# 📝 チャンピオン戦術プロトコル：アニビア (Anivia)

## 1. Core Profile
- **Champion**: Anivia (The Cryophoenix)
- **Role**: Mid
- **Power Spike**: Mid-Late Game (2コア完成以降)

## 2. Core Identity
氷壁（W）による陣形分断と、Ult（R）による制圧。集団戦の主導権を握り、敵を盤上の駒として支配せよ。

## 3. Timeline Strategy

### Early Game (0-15m)
- **ウェーブコントロール**: マナ枯渇は死を意味する。Qでスタンを確定させ、Eの追撃を叩き込む。この一連の動作に全リソースを注げ。
- **CS確保**: 弾速の遅い通常攻撃をタワー下で完璧に制御せよ。ここをミスるようでは、中盤のキャリーなど夢のまた夢だ。
- **生存**: パッシブの卵は保険ではない。序盤の卵化は即座に処刑を意味する。無理なトレードは厳禁。レベル6と女神の涙のスタックを最優先で積み上げろ。

### Mid Game (15-25m)
- **オブジェクト戦の支配**: ドラゴン・グラブ戦におけるRは、敵に対する「通行止め」の宣告だ。進路にRを敷き、Wで退路を封殺する。
- **リソース配分**: 青バフの確保は義務。マナを失ったアニビアは、ただの脆弱な鳥に過ぎない。

### Late Game (25m+)
- **エンドゲームプラン**: 敵キャリーを壁で孤立させ、味方と共に蒸発させる。火力を出すこと以上に、敵に「戦いたい場所で戦わせない」ことが至上命題。
- **ポジショニング**: 常に最後尾を維持せよ。アニビアの死は、チームの防衛力が霧散することを意味する。

## 4. Matchup Matrix

- **Favored**: 
    - **短射程メイジ/メレー（サイラス、カタリナなど）**: 突入の瞬間にQを置き、Wで退路を塞ぐ。彼らはただの凍った彫像と化す。
- **Countered**: 
    - **超長距離砲（ゼラス、ヴェル＝コズ）**: 射程外から削られるとゾーンコントロールは無効化される。耐え抜き、集団戦での一撃に全てを賭けろ。
    - **高機動アサシン（ゼド、フィズ）**: 卵を割られるリスクが極大。相手がスキルを吐き出すまでQを温存する、極限の忍耐が必要だ。
