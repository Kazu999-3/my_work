# 【Poppy Tactical Protocol】戦場を支配する「鉄槌」の運用理論

戦場における力とは、暴力の行使ではない。敵から「選択肢」という名の自由を奪うことだ。パッチ16.10においてポッピーは、環境に蔓延するハイパーキャリーや突撃型ブルファイターを完封する、唯一無二の「メタの門番」として君臨する。

知略なき武力は、霧の中を彷徨う愚行に過ぎない。我々の戦術指針をここに記す。

---

## 1. Core Profile
- Champion: Poppy
- Role: Jungle / Top
- Power Spike: Mid Game (15-25m)

## 2. Core Identity
「W『守護者の鉄槌』による機動性の完全否定と、壁際へのE『ヒロイックチャージ』による孤立キルの創出」
ポッピーの真価は、敵が描く「理想のプレイ」を物理的に拒絶する能力にある。

---

## 3. Timeline Strategy

### Early Game (0-15m)
Grubsの支配こそが最優先事項だ。ポッピーのタイマン性能と壁際へのノックバック能力は、ピット内での乱戦において圧倒的優位を約束する。レーン介入時は安易な壁ドンを急ぐな。Wを温存し、敵のブリンクスキルを誘発させた瞬間に叩き潰す。

### Mid Game (15-25m)
20分時点で1500gの優位を確保せよ。スカトル管理とカウンタージャングリングを徹底し、敵の生存圏を極限まで削る。集団戦におけるR『守護者の審判』は、敵の頭数を減らすためではなく、陣形を分断し、孤立した獲物を仕留めるために使うべきだ。

### Late Game (25m+)
後半、敵キャリーの脅威が増せば、役割を「盾」から「拒絶の壁」へと転換する。敵のバックラインへ飛び込むのは悪手。味方のADCに触れようとする敵ファイターを剥がし続けろ。敵が「何もさせてもらえない」という絶望を噛み締めたとき、勝利は確定する。

---

## 4. Matchup Matrix

### Favored
- Aatrox: ダッシュをWで完全に無効化可能。彼がQを振る前に位置を調整し、壁際へ押し込むだけでいい。その強気な攻撃性を、無力な沈黙へと変える。
- Dash-heavy Assassins: 突進スキルを持つ者は、ポッピーの前ではただの餌に過ぎない。

### Countered
- Zyra: 彼女の生成する植物はEの阻害要因となる。長射程ハラスに対し、安易な接近は自殺行為だ。視界スコアを毎分2.0以上維持し、不意のバーストを避けることを最優先せよ。

---

## 戦術家からの訓示

ジンクスの後半の爆発力を抑え、ザイラの序盤の強気を挫く。それが今のメタにおいてポッピーに求められる唯一の回答だ。

敵が動く時、そこには必ず隙が生まれる。その一瞬の迷いを鉄槌で叩き潰せ。

「動く者は、止まる運命にある。」

この言葉を心に刻み、戦場を支配しろ。
