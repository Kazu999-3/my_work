# 🚀 Discordサーバー構築クイックスタートガイド (KTM版)

詳細設計に基づき、以下の手順でサーバーを物理的に構築してください。

## 1. サーバーの作成
1. Discordの左側「＋」から「オリジナルの作成」を選択。
2. 名前を **『KTM : Keystone Tactics Match』** に。

## 2. カテゴリとチャンネルの設置
以下の4つのカテゴリを作成してください。

1.  **【!!】 初めに (Entrance)**
    - `#📍はじめに` / `#📝自己紹介` / `#📢案内板`
2.  **💬 【Main】 ラウンジ (Public Square)**
    - `#🌍広場` / `#🍻社会人の雑談` / `#📸スクショ自慢`
3.  **🎮 【Game】 対戦募集 (Matchmaking)**
    - `#募集-カスタム` / `#募集-ノーマル` / `#募集-ARAM`
4.  **🛡️ 【Tactics】 振り返り (Strategy)**
    - `#議論-振り返り` / `#質問-ビルド相談`

## 3. ロール（権限）の重要設定
ここがコミュニティの質を決める心臓部です。

1.  **デフォルト権限の設定**:
    - `@everyone` ロールの設定で「チャンネル閲覧」と「メッセージ送信」を **オフ** にします。
    - `#📍はじめに` と `#📝自己紹介` だけ「閲覧・送信可能」に個別設定します。
2.  **認可ロールの作成**:
    - `Member` (または `社会人`) というロールを作成します。
    - このロールに、雑談や募集チャンネルの「閲覧・送信権限」を与えます。
3.  **ランク・ポジションロール**:
    - 前述の設計案に基づき、色付きのロールを作成します。

## 4. 仕上げ：募集メンション用の設定
ロール作成画面で、`custom`, `normal`, `aram` の各ロールについて **「このロールへの@mentionを許可する」** を必ずオンにしてください。これにより、興味のある人だけを呼び出すことができます。
