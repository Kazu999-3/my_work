# ✍️ 高品質 note 制作手順書 (note_production_guide.md)

AIを活用し、1チャットの文脈（コンテキスト）を維持したまま、30分で高品質な note を書き上げるためのガイドラインです。

## 🧠 ステップ 1：一貫した文脈の維持（つなげるプロンプト）
別々のチャットに分けず、同じチャット内で以下の順序で対話を進めます。

1.  **ジャンル選定＆スコアリング**:
    HARMの法則、マズローの欲求段階、ジョブ理論を基に「今、最も売れるジャンル」を15個抽出し、スコアが高いものを1つに絞る。
2.  **ペルソナ深掘り**:
    ターゲットの「日々の不満」「深夜に一人で悩んでいること」「理想の未来」を極限まで具体化する。
3.  **本文構成（19の原則）**:
    読者の「買わない理由（時間がない、怪しい、自分には無理）」を一つずつ潰す構成案を作成。
4.  **本文執筆（5,000文字〜）**:
    これまでの分析をすべて踏まえた上で、読者の心に刺さる言葉で本文を生成。
5.  **セールスレター作成**:
    記事を売るための「導入部分」と「販売ページ」を、同じ熱量で作成。

## 🔥 ステップ 2：15分魂の編集（人間による仕上げ）
AIの出力をそのまま公開せず、以下の点に絞って手動で「熱量」を注入します。

- **体験の具体化**: AIが書いた一般論を、自分の「泥臭い実体験」に1〜2箇所差し替える。
- **NG表現の微調整**: `ng_words.md` を突き合わせ、自分の口癖や温度感に合わせる。
- **結論の断言**: 読者の背中を強く押すために、語尾を力強く（「～です」「～しましょう」）修正する。

## 📈 収益化スケジュール
- **無料 note (集客)**: X のプロフや固定リプから誘導。フォロワーを「ファン」に変える。
- **有料 note (販売)**: 尖った知識や、再現性のある手法を販売。資産として収益を積み上げる。
