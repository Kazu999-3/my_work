# 📊 重要指標管理 (KPI.md)

Antigravity の成長を測定するための重要指標（KPI）を管理します。毎週・毎月の分析レポートのベースとなります。

## 🎯 最優先指標 (North Star Metric)

- **月間収益**: 短期目標 5万円 / 中期目標 20万円

## 📈 追跡指標 (Tracking Metrics)
### X (Twitter)
- フォロワー数推移
- インプレッション合計
- プロフィールクリック率
- 有料noteへの誘導数

### Note
- 週間PV数
- スキ数
- 販売数・売上

## 📅 確認頻度
- **毎週日曜日**: 前週の数値を記録し、`reports/` に分析結果を出力する。
