# Knowledge 運用ガイド

このフォルダは、リサーチや日常の気づきを「資産」として蓄積する場所です。

## 階層構造
- **`raw/`**: 未加工のメモ、URLのダンプ、一次情報。
- **`domains/`**: 体系化された知識資産（ドメイン別）。
    - **`lol/`**: チャンピオン別戦術、パッチ分析。
    - **`automation/`**: プロンプトエンジニアリング、システム設計の知見。
    - **`monetization/`**: マーケティング、収益化戦略、X/note運用。
- **`reports/`**: 分析済みレポート、トレンド予測。

## アンちゃんへの指示
1.  リサーチした情報は、まず `raw` に保存するか、直接適切な `domains` 配下に構造化して記録してください。
2.  定期的に `raw` の情報を整理し、`domains` への昇格や `reports` への統合を行ってください。
