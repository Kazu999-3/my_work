# 🧪 実験フォルダ (experiments/)

本フォルダは、運用上のA/Bテストや試行錯誤の結果を蓄積し、データに基づいた「勝ちパターン」を抽出するための場所です。

## 1. 記録ルール

新しい実験を行う際は、`YYYYMMDD_実験名.md` という形式でファイルを作成し、以下の項目を記録してください。

- **目的**: 何を検証したいか（例：冒頭に【朗報】を付けると反応が上がるか）
- **仮説**: どうなると予想するか
- **内容**: A案とB案の具体的な内容
- **結果**: 1日/3日/7日後の数値（インプレッション、RT、成約数等）
- **考察**: なぜそうなったか、次からどうするか

## 2. 自動最適化サイクル

10個程度の実験データが溜まったら、アンちゃんに以下の指示を出してください。

> 「`experiments/` フォルダのデータを分析して、最も効果の高い『勝ちパターンのルールブック』を新設（または更新）して」

これにより、アンちゃんの運用戦略（`strategy.md`）や経営憲法（`ANTIGRAVITY.md`）が自律的に進化し続けます。
