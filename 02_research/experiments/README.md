# 🧪 実験・ABテスト管理 (experiments/)

このフォルダは、アンちゃんの運用を「感覚」から「データ」に基づくものへと進化させるための実験場である。

## 1. 記録すべき項目
実験を行う際は、以下の項目を含むレポートを `exp_YYYYMMDD_タイトル.md` として作成せよ。

- **仮説**: 何を狙って変化させるのか（例：深夜に投稿したほうがインプレが高いのではないか）
- **変数**: 変更した部分（例：投稿時間、プロンプトの口調、画像の種類）
- **結果**: 具体的な数値（インプレッション、クリック数、反応）
- **結論**: 次回からのルールに採用するかどうか

## 2. 実験のサイクル
1. **計画**: 過去の `wins.md` や `fails.md` から次の課題を特定。
2. **実行**: アンちゃんが新しいプロンプトや設定で稼働。
3. **分析**: ユーザーからのフィードバックやSNSの数値を元に分析。
4. **統合**: 成功パターンを `01_spirit/strategy.md` や `skills/` に反映。
