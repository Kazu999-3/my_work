# 🎯 ターゲットの悩み・不安 (pain_list.md)

以下は、ターゲットユーザー（例：副業初心者、AI活用に関心がある層）が抱える「痛み・悩み」のリストです。ここからAIが投稿のフックを生成します。

## ⚠️ 痛み（Pain）リスト
- [ ] AIをどう収益に結びつければいいかわからない
- [ ] 毎日何を発信すればいいか悩んで時間が溶ける
- [ ] プロンプトの書き方が難しくて挫折した
- [ ] フォロワーが増えてもマネタイズできる気がしない
- [ ] ツールばかり増えて使いこなせていない
- [ ] 他のインフルエンサーと何が違うのか説明できない
- [ ] 継続がとにかくしんどい
- [ ] 自分の経験が誰かの役に立つと思えない
- [ ] 専門知識がなくて「浅い」コンテンツしか作れない
- [ ] 自動化したいが、どこから手をつければいいか不明
