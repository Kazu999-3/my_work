# リサーチレポート：ザイラJG 徹底分析 (Patch 16.5)

## 1. 総合診断：現在の立ち位置
- **[Patch 16.5] 状況**: S〜Aティア級のポテンシャルを持つハイエンド・ジャングラー。
- **核心**: 「爆速のクリア速度」と「オブジェクト制覇力」が最大の武器。Agurin氏らが提唱する効率的パース（Agurin Path）の概念を最も体現できるチャンピオンの一つ。

## 2. テクニカル・多角的分析
### A. ビルド・アイテムの変遷
- **[Patch 16.5] コアアイテム**:
  - `Liandry's Torment`: 必須。プラントとの相性が極めて高く、継続的なダメージ源となる。
  - `Shadowflame`: 低HPターゲットへのトドメ性能が高い。
- **[Patch 16.5] 議論点**: `Rylai's Crystal Scepter` は状況次第。メレー相手には強力だが、既にスキルにスロウがあるため、ハストや魔貫（`Cryptbloom`）を優先する選択肢も有力。
- **Runes**: `Electrocute`（序盤のガンク強化）または `Dark Harvest`（後半のスケーリング）。

### B. マッチアップ（天敵・カモ）
- **[Patch 16.5] 有利**: `Jarvan IV`, `Vi` などの飛び込み系メレー。プラントでエンゲージを無効化しやすい。
- **[Patch 16.5] 不利**: `Karthus`（クリア速度で負け、スケールで抜かれる）、`Assassins`（不意打ちに弱い）、`Xerath/Viktor`（アウトレンジされる）。

### C. ルート・ジャングル技術
- **[Patch 16.5] 最速クリアの鍵**:
  - `Blue Sentinel` 到着時間を早め、パッシブの種を最大限に沸かせる。
  - `Wall-Hop Seeds`: レッドバフを叩きながら壁越しにクルーグへ種を植える技術。
  - `Multi-Camp Clearing`: 種を撒き、複数のキャンプを同時に削る。
- **[Patch 16.5] テンポ**: レベル3でのミッドガンク後、カニ（Scuttle）を制圧するルートが強力。

### D. 海外（Reddit/英Wiki）の深層情報
- **Agurin Path の応用**: Agurin氏が提唱する「効率的フルクリア（Agurin Path）」の概念は、ザイラJGの爆速クリアと極めて相性が良い。現在のシーズン16においても、この思考ロジックをベースにしたザイラ運用がハイエロで推奨されている。
- **プラントによる視界確保**: 深い位置への種植えによるリスクヘッジ。

## 3. プレイ改善への直言
- 「キャンプをミリ残しして次のキャンプへ移動する」技術を徹底せよ。プラントが最後の一撃を入れてくれる。
- 敵のCCをプラントで受ける（スキルブロック）練習を積むことで、対面有利を覆せる。

## 4. note/資産化：独自の切り口
- **「Agurin式：思考停止フルクリアのその先」**: ただ速いだけでなく、その速さをどう「敵の絶望」に変えるかの戦略。
- **「脱リライ論」**: 統計に騙されない、真のアイテムビルド選択。
