#### 🔥 「脱リライ論」の深掘り

> 高エロ帯（Master+）のRedditで繰り返し議論されるテーマ。

- **リライ肯定派**: メレー多め構成で効果絶大。プラントの一撃ごとにスロウが入り、逃げ切れない状況を作れる。
- **リライ否定派**: ザイラのEにすでにルート、プラント（Vine Lasher）にもスロウがある。赤スマイトのスロウも加味すると、追加スロウは過剰。**魔法貫通（Void Staff）やAccuracy Haste（Cryptbloom）のほうが実ダメージ貢献が大きい**。
- **結論**: 敵の構成を見て判断。メレー3体以上→リライ、それ以外→Void StaffまたはMorelloが最適解。

### C. ルート＆ジャングルクリア技術

#### 最速フルクリア（2:50以下目標）

**ルートA（青側スタート推奨）**:
```
Blue → Gromp → Wolves → Raptors → Red → Krugs → Gank or Scuttle
```

**ルートB（ラプター起点・攻撃的）**:
```
Raptors → Red → Krugs → リコール（Dark Seal購入） → Blue側攻めor対面JG
```

#### 🔑 クリア最適化のクリティカル要因

1. **種の事前配置**: キャンプに到着する前にパッシブの種が沸くタイミングを計算し、Wで追加配置。**Blue Sentinelには1秒でも早く到着**して種を湧かせる。
2. **壁越し種植え**: レッドバフを叩きながら壁越しにクルーグ方面へ種を配置→Qで起動して同時削り。
3. **ミリ残し移動**: キャンプをミリ残しにして次のキャンプへ移動。プラントが最後の一撃を代行してくれるため、移動時間を稼げる。
4. **2番目のプラント遅延生成**: Wで生んだプラントは2体目に50%ダメージ軽減が入る。**同時に出さず、1体目のプラントがダメージを出し終えてから2体目を出す**のが最適解。
5. **スマイト温存**: ザイラはスマイト1回でフルクリア可能。2回目のスマイトはScuttleまたはガンク後のヘルス回復に温存。

- **[Source]**: [Reddit r/zyramains](https://reddit.com), YouTube VOD各種, [jungler.gg](https://jungler.gg)

## 5. プレイ改善：マクロ＆ミクロ

### ミクロ（操作技術）
1. **スキルブロック**: 敵のCCスキルをプラントで受ける。特にBlitzのQ、Threshのフックなど、ザイラ本体に当たれば致命的なスキルをプラントが吸収。
2. **種の壁越し配置**: 赤バフ/ドラゴン周辺の壁越しにWで種を植えることで、視界確保＋クリア最適化を同時に行う。
3. **R→Zhonya's コンボ**: 集団戦でR→即ストップウォッチ。ノックアップ発動時にはプラントが敵を削り、自分は安全。

### マクロ（判断力）
1. **レベル3ミッドガンク**: フルクリアルートAの後、Scuttle前にミッドへガンク。E→Q→Wのコンボで高確率のキルorフラッシュ消費。
2. **オブジェクト制圧**: ドラゴン/ヘラルドはプラント火力で高速処理。敵JGが姿を見せた瞬間に即ソロ。
3. **カウンターJG**: クリア速度で勝つ相手（ほぼ全員）に対して、相手の2周目ジャングルを奪いに行く。
