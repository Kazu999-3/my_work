# KTM Bot 拡張計画: チーム分け自動化プロトコル

## 1. プロジェクト概要
Discordでのカスタム募集からチーム分け、スプレッドシートへのデータ反映までを完全に自動化し、管理者の手動操作をゼロにする。

## 2. MVP（最少構成）の範囲
- 募集枠（10名）が埋まった瞬間にトリガーを起動。
- 参加確定した10名の名前を「対戦入力」シートのA列に即時反映。
- 続けてチーム分け（バランサー）を実行し、シートを更新。
- 結果を Discord に自動投稿し、メンションで通知。

## 3. 技術スタック
- **Backend**: Google Apps Script (GAS)
- **Frontend/Middleware**: Cloudflare Workers (Discord Bot)
- **Database**: Google Sheets

## 4. 現在のステップ
1. [x] 募集完了時の自動リクエスト実装 (Discord側)
2. [x] 自動チーム分けハンドラの調整 (GAS側)
3. [/] 募集内容の編集機能の実装 (Discord側)
4. [ ] 動作確認（味見）

## 5. 運用ルール
- 破壊的変更（既存データの削除等）を行う前には必ず `y/n` で確認を取る。
- エラー発生時は「手動実行」にフォールバックできる仕組みを維持する。
