# 🩹 自己修復プロポーザル

承知いたしました。アンちゃんズ AdminAn (System Architect) として、自律商社システムのメインサイクルで発生した `AttributeError: 'OmniAgent' object has no attribute 'harvest_assets'` エラーについて分析し、修正案を提示します。

---

### エラー分析

1.  **エラーメッセージ:** `'OmniAgent' object has no attribute 'harvest_assets'`
2.  **発生箇所:** `omni_agent.py` の `run_cycle` メソッド内の145行目、`self.harvest_assets()` の呼び出し。
3.  **原因:** `OmniAgent` クラスの定義内に `harvest_assets` という名前のメソッドが実装されていません。しかし、`run_cycle` の実行フローにはその呼び出しが含まれているため、未定義メソッドの呼び出しにより `AttributeError` が発生しています。

### `harvest_assets` メソッドの意図推測

エラー発生箇所の前後にあるログメッセージと、既存のメソッド `save_draft` および `assemble_products` の処理内容から、`harvest_assets` メソッドの役割は以下のように推測されます。

*   **ログメッセージ:** `self.log("🧺 [AdminAn] Step 9/12: 高評価ドラフトの資産化...")`
*   **関連メソッド `save_draft`:** `03_factory/daily_posts` ディレクトリにドラフトを `draft_YYYYMMDD_HHMM_sSCORE.md` の形式で保存している。
*   **関連メソッド `assemble_products`:** `01_spirit/assets` ディレクトリから `evergreen_*.md` 形式のファイルを読み込んでプロダクトを組み立てている。

これらの情報から、`harvest_assets` は **`03_factory/daily_posts` にある高スコア（例: 80点以上）のドラフトを、永続的な資産として `01_spirit/assets` ディレクトリに移動・リネームする** 役割を果たすものと判断できます。特に「資産収穫」という言葉から、一時的なドラフトから永続的な資産への「移動」が自然です。

### 修正案（Pythonコード）

`OmniAgent` クラスに `harvest_assets` メソッドを追加し、上記の推測に基づいた処理を実装します。ファイル操作のため `shutil` モジュールもインポートします。

    def harvest_assets(self):
        """
        [AdminAn] 高評価のドラフトを永続的な資産 (01_spirit/assets) として収穫する。
        スコアが閾値（デフォルト80点）以上のドラフトを移動し、ファイル名を evergreen_YYYYMMDD_HHMM.md に変更する。
        """
        self.log("🧺 [AdminAn] 高評価ドラフトの資産化プロセスを開始します...")
        
        draft_dir = ROOT_DIR / "03_factory" / "daily_posts"
        asset_dir = ROOT_DIR / "01_spirit" / "assets"
        # ensure_dirs で作成済みだが念のためここでも確認
        asset_dir.mkdir(parents=True, exist_ok=True) 

        harvested_count = 0
        score_threshold = 80 # 資産化するスコアの閾値

        for draft_file in draft_dir.glob("draft_*.md"):
            try:
                # ファイル名からスコアを抽出 (例: draft_20231027_1030_s92.md)
                parts = draft_file.stem.split('_s') # 'draft_YYYYMMDD_HHMM', 'SCORE'部分
                if len(parts) == 2 and parts[1].isdigit():
                    score = int(parts[1])
                    
                    if score >= score_threshold:
                        # 新しいファイル名を生成 (スコアを除去し、evergreen_プレフィックスを付与)
                        # 例: draft_20231027_1030 -> evergreen_20231027_1030.md
                        original_timestamp_part = parts[0].replace('draft_', '')
                        new_filename = f"evergreen_{original_timestamp_part}.md"
                        destination_path = asset_dir / new_filename
                        
                        # ファイルを移動
                        shutil.move(str(draft_file), str(destination_path))
                        self.log(f"✅ 高評価ドラフトを資産化しました: '{draft_file.name}' -> '{destination_path.name}'")
                        harvested_count += 1
                    else:
                        self.log(f"ℹ️ ドラフト '{draft_file.name}' は資産化の閾値({score_threshold}点)に達しませんでした (スコア: {score})。")
                else:
                    self.log(f"⚠️ ドラフト '{draft_file.name}' のファイル名からスコアを抽出できませんでした（スキップ）。")
            except Exception as e:
                self.log(f"❌ ドラフト資産化中にエラーが発生しました ('{draft_file.name}'): {e}")

        if harvested_count == 0:
            self.log("💡 資産化すべき高評価ドラフトは見つかりませんでした。")
        else:
            self.log(f"🧺 [AdminAn] {harvested_count}個のドラフトを資産として収穫しました。")

                         self.log(f"⚠️ ステータス更新はスキップされました（環境依存）: {u_res.status_code}")
             except Exception as e:

    def audit_draft(self, draft):
        """Auditor: ドラフトを検閲し、スコアリングする"""
        style_dir = ROOT_DIR / "01_spirit" / "style"
        prohibited = (style_dir / "prohibited_patterns.md").read_text(encoding="utf-8") if (style_dir / "prohibited_patterns.md").exists() else ""

以下のドラフトを厳格に監査し、100点満点で採点してください。

【採点基準】
1. NGワード( {ng} ) または スタイル禁止事項( {prohibited} ) が1つでも含まれていたら 0点。
2. 日本語として不自然、またはAI丸出しの定型句があれば大幅減点。
3. 戦略的フックが弱く、読者がスルーしそうな内容なら減点。

必ず以下のJSON形式でのみ回答してください。余計な文章やMarkdownの枠（```jsonなど）は一切不要です。
{{"score": 85, "feedback": "具体的な不足点や修正すべき箇所..."}}

【監査対象ドラフト】
{draft}
"""
        # 監査は Lite で実行
        res_text = generate_with_fallback(prompt, preferred_model=self.model_lite)
        try:
            import re
            # JSON部分をより柔軟に抽出
            match = re.search(r'\{.*\}', res_text, re.DOTALL)
            if match:
                clean_json = match.group(0).replace('\n', '').replace('\r', '')
                data = json.loads(clean_json)
                return int(data.get("score", 50)), data.get("feedback", "理由なし")
            return 50, f"JSON形式が見会えませんでした: {res_text[:50]}..."
