## エラー分析と修正案

### エラー内容の分析

提供されたエラーメッセージ `'OmniAgent' object has no attribute 'harvest_assets'` とトレースバックから、`OmniAgent` クラスのインスタンスにおいて、`harvest_assets` というメソッドが存在しないにも関わらず、`run_cycle` メソッド内で `self.harvest_assets()` が呼び出されていることが原因で発生しています。

ソースコードを確認すると、`run_cycle` メソッドの後半に以下の行があります。

```python
            # 9. AdminAn: 資産収穫 (Asset Harvesting)
            self.log("🧺 [AdminAn] Step 9/12: 高評価ドラフトの資産化...")
            self.harvest_assets()
```

この部分で `harvest_assets` メソッドが呼び出されていますが、`OmniAgent` クラスの定義内には `harvest_assets` というメソッドは定義されていません。

### 修正案

このエラーを解決するには、以下のいずれかの方法で対応します。

#### 修正案 1: `harvest_assets` メソッドの定義を追加する

`OmniAgent` クラス内に、以下のメソッド定義を追加してください。
