### 2. 勝率ブレイクポイントの真実（表面的なインサイトの提供）

*   **絶望の序盤 (0-15分) - なぜ弱いのか？**
    ムンドの序盤はまさに「絶望」だ。この時間帯の勝率は著しく低い。ジャングルクリア速度は遅く、Q(汚染肉斬り包丁)のヘルスコストが重い。Gankを仕掛けてもダメージ不足でキルに至りにくく、敵ジャングラーとの遭遇戦ではほぼ確実に負ける。
    現代ジャングルでは、この序盤でどれだけ早くオブジェクト（スカトル、ドラゴン、ヘラルド）を確保し、Gankでレーンに影響を与えられるかが非常に重要視される。ムンドは序盤のGank性能もデュエル能力も低いため、この高速テンポについていけない。無理なGankで時間とヘルスを浪費したり、インベードでキャンプを荒らされたりすれば、そのままゲームエンドまで立て直すのが困難になる。

*   **我慢の中盤 (15-25分) - なぜ回復し始めるのか？**
    初手アイテム（例: ストライドブレイカー）や防御アイテムが1〜2つ完成する頃から、勝率は徐々に回復し始める。W(心臓震盪)の回転率が上がり、R(最大投与)の回復量が体感できるレベルになるためだ。この時間帯から、ドラゴンやヘラルドを巡る小規模な集団戦で、フロントラインとして味方を守りつつ、持続ダメージとCCで貢献できるようになる。
    しかし、この「我慢」の期間はまだムンドが本領を発揮できるわけではない。敵も回復阻害アイテムを揃え始めるため、Rの使用タイミングには細心の注意が必要だ。

*   **覚醒の終盤 (25分以降) - なぜ覚醒するのか？**
    コアアイテムが3つ以上完成すると、ムンドは「覚醒」する。ヘルスと防御力が圧倒的なレベルに達し、Rの回復量が凄まじいことになり、勝率が最も高くなる時間帯だ。敵のCCが少ない構成であれば、まさに止められない存在となる。

### 3. トッププレイヤーの暗黙知（秘匿インサイト）
ムンドでD Tierを抜け出すには、表面的な知識だけでなく、トッププレイヤーが実践する「知らないと負ける」レベルのミクロテクニックが必須だ。ここでは、ジャングルクリア速度の最適化、スキルコスト管理、そしてGankの選択肢に直結する秘匿技術を具体的に解説する。

*   **Q(汚染肉斬り包丁)の精密制御術 - 「どう操作する？」**
    *   **ジャングルクリア**: 小型キャンプのミニオンにはQを使わず、大型モンスターにのみ使用しろ。これによりヘルスコストを最小限に抑えつつ、効率的にダメージを与える。Qを外すことはヘルスの浪費だけでなく、クールダウン中の貴重なダメージソースを失うことにも繋がる。常に確実なヒットを心がけろ。
    *   **壁越しのGank/視野外エンゲージ**: 特定の薄い壁越しにQを投げることで、敵のワードを回避し、視野外からスローを入れてGankのセットアップをすることが可能だ。ミッドレーンの両サイドの壁や、ドラゴンピット裏の壁などが狙い目。この奇襲で敵の意表を突き、序盤のGank成功率をわずかでも高めるのだ。

*   **W(心臓震盪)の回転率と活用術 - 「どう操作する？」**
    *   **ヘルス管理**: Wは発動中にダメージを受けると、そのダメージの一部を回復する。特に序盤のジャングルクリアでは、この回復効果を最大限に活用し、ヘルスを維持しながらファームするのだ。効果時間のギリギリまで敵からのダメージを受け続け、再発動で追加ダメージとノックバックを与えるのが理想だ。
    *   **オブジェクトスティール/妨害**: エルダーやバロンといったオブジェクトスティールの際、Wの再発動によるノックバックを、敵ジャングラーがスマイトを打つ瞬間に合わせることで、スマイトをキャンセルさせたり、敵チャンピオンをオブジェクトから引き離したりできる可能性がある。これは練習が必要だが、成功すればゲームをひっくり返す「一発逆転」の秘策となる。

*   **E(ぶっ飛ばし)の攻撃速度リセットとタワーダメージ - 「どう操作する？」**
    *   **クリア速度向上**: 通常攻撃の間にEを挟むことで、AAタイマーをリセットし、瞬間的なバーストダメージを出す。「通常攻撃 → E → 通常攻撃」のコンボをジャングルクリアやタワーへのダメージに活用し、数秒単位で速度を短縮するのだ。
    *   **タワーダイブ**: Eはタワーにも追加ダメージを与えるため、タワーダイブ時には積極的に使用し、タワーを迅速に破壊して敵に反撃の機会を与えない。

*   **ファームパスとインベード耐性 - 「どう立ち回る？」**
    *   **基本フルクリア**: 序盤のGankが弱いMundoは、赤バフ → クルーグ → ラプター → ウルフ → 青バフ → グロンプのフルクリアを基本とし、最速でレベル4を目指せ。これにより、最低限のステータスとスキルポイントを確保し、次のプレイへの準備を整える。
    *   **インベード対策**: 敵が序盤から強いジャングラー（リー・シン、ニダリーなど）の場合、赤バフ側にワードを置き、インベードを警戒しろ。もしインベードされたら、無理にファイトせず、逆サイドのジャングルをファームするなどして、経験値とゴールドの差をつけられないように立ち回る。

### 4. メタジャングラー対策（具体的な立ち回り）
パッチ14.10のメタは、序盤のGank圧力、速いクリア、オブジェクトへの迅速な関与を重視している。この環境下でドクター・ムンドは苦戦を強いられるが、相手の構成やプレイスタイルを見極めることで、活路を見出すことができる。

*   **絶対に避けるべき相手 (明確なカウンター) - 「誰を避ける？」**
    *   **回復阻害を持つチャンピオン、または回復阻害アイテムを積む構成**:
        *   **例**: ケイトリン、ミス・フォーチュン、ヴェイン（ADC）、アッシュ（サポート）、モレロノミコンやケミテックピュートリファイアを積むAP/サポート。
        *   **具体的な立ち回り**: ムンドのRによる莫大な回復は、回復阻害効果で容易に無効化される。彼らが育つ前にレーンGankを成功させ、圧力をかけることが重要だ。集団戦では、彼らを真っ先にフォーカスするか、回復阻害が入る前にRを使い切る立ち回りが求められる。不可能であれば、彼らがサイドレーンにいる間にオブジェクトを取るなど、直接対決を避けるマクロプレイに徹しろ。
    *   **序盤のデュエルが強力なジャングラー**:
        *   **例**: リー・シン、エリス、ニダリー、ヴァイ、カイ＝サ（ジャングル）。
        *   **具体的な立ち回り**: 彼らは序盤のGank性能、デュエル能力、クリア速度のいずれか、または全てにおいてムンドを上回る。序盤に積極的にワードを置き、敵ジャングラーの位置を把握しろ。彼らが逆サイドにいる間に、自分のジャングルをファームするか、オブジェクトを狙う。敵ジャングルに無理に踏み込まず、安全なプレイを心がけろ。

*   **積極的に狩るべき相手/有利な立ち回り - 「誰を狩る？」**
    *   **ファーム系ジャングラー/序盤のデュエルが弱いジャングラー**:
        *   **例**: マスター・イー、カ＝サンテ（ジャングル）、アイバーン。
        *   **具体的な立ち回り**: これらのチャンピオンはムンドと同様に序盤が弱かったり、Gank圧力が低かったりするため、Mundoが有利にファームを進められる。中盤以降、コアアイテムが揃い始めたら、彼らのジャングルに侵入してキャンプをスチールし、ゴールドと経験値の差を広げろ。彼らがGankに行った逆サイドでオブジェクト（ドラゴン、ヘラルド）を確保するのだ。
    *   **CCが少ない敵チーム構成**:
        *   **具体的な立ち回り**: ムンドのRは回復量が多い一方で、CCに弱い。敵チームにハードCC（スタン、ノックアップ、サプレッションなど）が少ない場合、ムンドはRを発動すれば止められることなく暴れまわることができる。このような構成の敵に対しては、終盤の集団戦で積極的にエンゲージし、フロントラインとして敵のダメージを吸い尽くせ。回復阻害が入らない限り、圧倒的な耐久力でチームを勝利に導ける。
