# AI臭さ排除：禁止表現ガイド (prohibited_patterns.md)

読者の「またAI記事か」という拒絶反応（重力）を回避するための禁止リスト。

## 1. 絶対禁止（即リジェクト）
| 禁止表現 | 理由 | 言い換え案 |
| :--- | :--- | :--- |
| **「いかがでしたでしょうか」** | AI丸出しの典型的な結び。 | 「さて、この仕組みをどう使いこなしますか？」 |
| **「〜を徹底解説！」** | 扇情的で中身が薄い印象を与える。 | 「〜の裏側にある設計思想を共有します。」 |
| **「要チェックです」** | 指示的で押し付けがましい。 | 「ここが、勝敗を分ける急所です。」 |
| **「まとめとして」** | 読者の離脱を招く。 | 「ここまでの知見を、一つの武器に統合しましょう。」 |

## 2. 構文の禁止
- **「まず、次に、最後に」の多用**: 教科書的な説明は読み飛ばされる。エピソード形式で語れ。
- **「過剰な箇条書き」**: 箇条書きは結論のみ。考えのプロセス（泥臭い試行錯誤）は文章で。

## 3. 精神性の禁止
- **「ポジティブすぎる未来」**: 根拠のない楽観は信頼を削る。現実的なリスクと対策をセットで語れ。
