# AI臭さ排除：禁止表現ガイド (prohibited_patterns.md)

読者の「またAI記事か」という拒絶反応（重力）を回避するための禁止リスト。

## 1. 絶対禁止（即リジェクト）
| 禁止表現 | 理由 | 言い換え案 |
| :--- | :--- | :--- |
| **「いかがでしたでしょうか」** | AI丸出しの典型的な結び。 | 「さて、この仕組みをどう使いこなしますか？」 |
| **「〜を徹底解説！」** | 扇情的で中身が薄い印象を与える。 | 「〜の裏側にある設計思想を共有します。」 |
| **「要チェックです」** | 指示的で押し付けがましい。 | 「ここが、勝敗を分ける急所です。」 |
| **「まとめとして」** | 読者の離脱を招く。 | 「ここまでの知見を、一つの武器に統合しましょう。」 |

## 2. 構文の禁止
- **「まず、次に、最後に」の多用**: 教科書的な説明は読み飛ばされる。エピソード形式で語れ。
- **「過剰な箇条書き」**: 箇条書きは結論のみ。考えのプロセス（泥臭い試行錯誤）は文章で。

## 3. 精神性の禁止
- **「ポジティブすぎる未来」**: 根拠のない楽観は信頼を削る。現実的なリスクと対策をセットで語れ。

| 禁止表現 | 理由 | 言い換え案 |
|---|---|---|
| やあ、みんな。 | AIが生成しがちな親しみやすいトーンで、自然な人間らしさを欠き、AI臭さを強調してしまう。 | （冒頭の挨拶を削除、または文脈に合わせた自然な導入） |
| 今日も元気に～ | 同上 | （文脈に合わせた自然な導入） |
| ～かい？ | 同上 | ～できているだろうか？、～してほしい、～しているか？ |
| ～と言えるでしょう | NGワードレベル1ではないが、より自然な表現への置き換えが望ましい。 | ～だ。、～と考えられる。 |
| ～してみてください | 同上 | ～してみてほしい。、～することをおすすめする。 |
| ～お役に立てれば幸いです | 同上 | ～となれば幸いです。、～で、お力になれれば。 |
| ～だと思われます | レベル2の表現であり、より断定的な、あるいは経験に基づいた語り口に修正することで、信頼性と個性を高めることができる。 | ～だ。、～だろう。、～だと確信している。 |
| ～せよ | 指示的で押し付けがましい印象を与え、AI臭さ排除：禁止表現ガイドの「指示的で押し付けがましい」という理由に抵触する可能性がある。 | ～してほしい。、～することをおすすめする。、～を検討しよう。 |
| 羨望マーケティングの極意 | やや扇情的で、中身が薄い印象を与える可能性がある。 | （より具体的で、内容を正確に表す表現へ修正） |
