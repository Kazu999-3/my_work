# 勝利法則 (wins.md)

アンちゃんが自律的に発見・継承した、収益に直結する黄金パターン。

## 1. 商品ラダーによる価値の最大化

単発の切り売りではなく、以下の階段を読者に登らせる。
- #認知 #トレンド : トレンドに乗り、期待値を超える情報を X/Thread で無料で提供。 (Free)
- #教育 #価値提供 : 有料コンテンツを通じて「この人の話は本物だ」と確信させる。 (Value Up)
- #資産化 #スキルパック : 自分が使っている「AIスキル（プロンプト等）」をそのままパッケージ化して共有。
- #仕組み提供 #フルサポート : 最終的に「システムそのもの」に触れてもらう。

## 2. コンテンツの心理的トリガー

- #脱労働 #メッセージ : 「AIを使って楽をしよう」ではなく「AIを使って労働から抜け出そう」というメッセージ。
- #逆張り #独自権威 : 「Google Antigravity」「Codex」など、一般層が知らない上位概念を提示。
- #証明 #解像度 : 抽象論ではなく、実際に稼働している「仕組みの解像度」を晒す。

## 3. テクニカル・ウィン

- #トレンド先取り : Lolalytics の勝率データ（??.5%）を、単なる数字ではなく「今すぐ行動すべき理由」として翻訳。
- #AI臭の排除 : 自己紹介や Vibe を語る際は、論理よりも感情（情熱、葛藤）を優先して記述。

## 4. 羨望マーケティング (Envy Strategy)
- #実績の晒し方 : 「稼げる」と言うな。「AIが勝手に仕事をしているので、自分は今〇〇で遊んでいます」という**ライフスタイルの落差**を晒せ。
- #導線の美学 : ツール単体ではなく、「これがこう繋がって、あそこで収益化される」という**全体図（インフラ）のパズル**を見せ、知的好奇心と羨望を同時に刺激。
