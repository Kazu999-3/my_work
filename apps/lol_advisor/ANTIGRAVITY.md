# 👑 LoL AI Advisor: "Project Antigravity"

このプロジェクトは、League of Legends (LoL) のプレイヤー（あなた）に対し、AI（アンちゃん）が Discord を通じて戦術的なアドバイス、試合分析、ピック支援を提供するものです。

## 1. 目的 (Why)
- データの裏付けに基づいた客観的なアドバイスで勝率を向上させる。
- 試合後の振り返りを自動化し、改善点を明確にする。
- ピック画面での迷いをなくし、最適な選択を支援する。

## 2. ターゲット & 利用シーン
- ランク戦で上を目指したいプレイヤー。
- 試合の合間やピック画面で、サッと統計情報を確認したい時。
- 自分の成長をデータで実感したい時。

## 3. 技術スタック (How)
- **言語**: Python 3.10+
- **ライブラリ**: `discord.py`, `requests`, `python-dotenv`
- **外部API**: Riot Games API (RGAPI), (将来的に) 統計サイトスクレイピング
- **AIエンジン**: Google Gemini API (分析・アドバイス生成用)

## 4. MVP (最小機能) の範囲
- **フェーズ1**: 
    - Discord Bot の基礎構築。
    - `/stats [GameName#Tag]` : Riot ID から基本プロフィール・ランクを取得。
    - `/counter [Champion]` : 指定したキャラの基本対策（手動/固定データから開始）を表示。
- **フェーズ2**: 
    - `/recent` : 直近の試合結果を取得し、簡易的なスコア判定を行う。
    - ピック画面での推奨BAN出力。
- **フェーズ3**:
    - Riot API のタイムラインデータを用いた「スノーボール分析」。
    - LCU API との連携による自動ピック検知。

## 5. 現在のステップ
1. Discord Bot の雛形作成と環境変数 (Riot API Key) の設定。
2. Riot API を叩いて自分自身のデータを取得してみる。
