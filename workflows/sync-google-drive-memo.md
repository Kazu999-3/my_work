# Googleドライブ・メモ同期ワークフロー

このワークフローは、iPhoneやPCからGoogleドライブの「AI連携メモ」フォルダに保存されたコンテンツ（テキスト、PDF等）を、AIが自律的に読み取り、現在のプロジェクトに統合するための手順を定義したものです。

## 1. フォルダの特定とリスト取得
// turbo
1. Googleドライブのマウントポイント（通常は `G:\マイドライブ\AI連携メモ`）を確認し、中身をリストアップします。
   ```powershell
   ls "G:\マイドライブ\AI連携メモ"
   ```

## 2. 対象ファイルの選別
2. 更新日時が最新のもの、またはユーザーが指定したファイル名（例：`今日のメモ.txt`）を特定します。

## 3. コンテンツの読み込み
3. ファイル形式に応じて以下の処理を行います。
   - **.txt / .md の場合**: 直接 `view_file` で内容を全件読み込みます。
   - **.pdf の場合**: 内部のテキスト情報を抽出して読み込みます。

## 4. スキルとの連携
4. 読み取った内容を以下のいずれかのスキルに投入します。
   - **記事作成**: `UltimateXNoteFunnelBuilder.md` に投入し、1週間分のX/note構成を生成。
   - **プロンプト化**: `UltimatePromptOptimizer.md` に投入し、メモを完璧なプロンプトに変換。
   - **分析**: `CompetitorAnalyst.md` 等に投入し、リサーチデータとして活用。

## 5. 実行のトリガー
ユーザーが「Googleドライブのメモを読んで」または「最新のメモから記事を書いて」と指示した際に、このワークフローを自動的に発動させます。
