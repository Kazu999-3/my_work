# 🚀 収益化note量産ワークフロー（500円モデル）

このワークフローは、Geminiの力を最大限に活用し、安定して読まれる「低単価note」を最速で作成・パッケージングするための手順です。

## 📋 事前準備
- [ ] `knowledge/reports/` 内の最新のトレンドレポート、または `knowledge/memo/` 内の特定のメモを一つ選ぶ。

## 🛠️ ステップ詳細

### 1. 構成とターゲットの決定
// turbo
1. AIに対し、選んだネタに基づいた「売れやすい構成案」を5つ提案させる。
   - 例: 「共感 → 問題提起 → 解決策 → 具体例 → 行動」
2. 最も「価値提供」が明確なものを1つ選び、ペルソナ（誰に捧げるか）を確定させる。

### 2. 本文の執筆
// turbo
1. 確定した構成に基づき、3,000文字〜5,000文字程度の記事執筆を指示する。
   - 指示のコツ: 「抽象論だけでなく、今日からできるアクションプランを最低3つ含めて」
2. アンちゃんが生成した下書きを確認し、必要に応じて事実確認や独自の体験談を追記する。

### 3. タイトルと見出しの磨き込み
// turbo
1. クリック率（CTR）を最大化するタイトル案を10個生成させる。
2. 読者がスクロールを止められないような「魅力的な小見出し」にリライトする。

### 4. パッケージングと価格戦略
1. 収益化戦略に基づき、無料部分と有料部分の境界線を決める（最も重要なノウハウを有料へ）。
2. 基本価格を500円（またはトレンドに合わせた戦略価格）に設定する。

### 5. アイキャッチと宣伝の自動生成
// turbo
1. `generate_image` ツールを使用し、内容にマッチした「プレミアム感のある表紙画像」を作成する。
2. X（Twitter）スレッド形式の宣伝文（4〜5投稿分）を自動生成する。

## 🎯 期待される成果
- 読者が納得して500円を支払える、高品質な実用系コンテンツ。
- 記事公開と同時にSNSで拡散を開始できる完璧な宣伝セットの完成。
