#  Jungle Pathing Protocol v1.0

## 1. Optimal Routes
- **Full Clear (Standard)**: Blue -> Gromp -> Wolves -> Raptors -> Red -> Krugs -> Scuttle.
- **Level 3 Rush**: Buff -> Buff -> Gromp -> Gank.

## 2. Objective Control
- 5:00 - Grubs spawning, ensure mid/top priority.
- 8:00 - Herald prep, deep vision in enemy jungle.

## 3. Counter Jungle Logic
