#  Vayne Tactical Protocol v1.0

## 1. Core Profile
- **Champion**: Vayne
- **Role**: Adc / Top
- **Power Spike**: Mid-Late (2-3 Items)

## 2. Core Identity
- 「シルバーボルト (W)」による割合ダメージと、「タンブル (Q)」による位置取りで、敵フロントラインを溶かしつつ生存し続けよ。

## 3. Timeline Strategy
### Early Game
- 射程の短さをカバーしつつ、CSを確実に確保。Eによる拒絶を徹底。
### Mid Game
- 「ファイナルアワー (R)」発動時のステルスを利用し、敵のスキルを回避してキルを奪え。

## 4. Matchup Matrix
- **Favored**: Tanks (Sion, Cho'Gath) - 逃げ場なき真ダメージ。
- **Countered**: Poke / Burst (Caitlyn, Draven) - 接近前の被弾に注意。
