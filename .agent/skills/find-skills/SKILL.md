# スキルを探す (Find Skills)

このスキルは、オープンなエージェントスキル・エコシステムからスキルを発見し、インストールする手助けをします。

## このスキルを使用するタイミング

ユーザーが以下のようなアクションをとった場合に使用してください：

- 「～をしたいが、どうすればいいか」という質問（一般的なタスクで既存のスキルがありそうな場合）
- 「～のスキルを探して」「～のためのスキルはあるか」と明示的に依頼された場合
- 「～はできるか」と聞かれ、それが専門的な能力である場合
- エージェントの機能を拡張することに興味を示した場合
- ツール、テンプレート、ワークフローを探したい場合
- 特定のドメイン（デザイン、テスト、デプロイなど）で助けが必要だと述べた場合

## Skills CLI とは？

Skills CLI (`npx skills`) は、オープンなエージェントスキル・エコシステムのパッケージマネージャーです。スキルは、エージェントに専門的な知識、ワークフロー、ツールを追加するモジュール式のパッケージです。

**主要なコマンド:**

- `npx skills find [query]` - キーワードでスキルを検索（対話形式または直接検索）
- `npx skills add <package>` - GitHubなどのソースからスキルをインストール
- `npx skills check` - スキルの更新を確認
- `npx skills update` - インストール済みの全スキルを更新

**スキルを探す（ウェブサイト）:** [https://skills.sh/](https://skills.sh/)

## ユーザーがスキルを見つけるのを支援する方法

### ステップ 1: ニーズを理解する

ユーザーが助けを求めたとき、以下を特定します：

1. **ドメイン** (例: React, テスト, デザイン, デプロイ)
2. **具体的なタスク** (例: テストの作成, アニメーション作成, PRレビュー)
3. **スキルの有無** (既存のスキルが存在する可能性が高い一般的なタスクかどうか)

### ステップ 2: まずリーダーボードを確認する

CLIで検索する前に、[skills.sh リーダーボード](https://skills.sh/) をチェックして、そのドメインに有名なスキルがないか確認しましょう。リーダーボードにはインストール数順にランク付けされた、実績のあるスキルが掲載されています。

例（Web開発のトップスキル）:
- `vercel-labs/agent-skills` — React, Next.js, Webデザイン (各10万以上のインストール)
- `anthropics/skills` — フロントエンドデザイン, ドキュメント処理 (10万以上のインストール)

### ステップ 3: スキルを検索する

リーダーボードで解決しない場合は、検索コマンドを実行します：

```bash
npx skills find [query]
```

例：
- 「Reactアプリを高速化するには？」 → `npx skills find react performance`
- 「PRレビューのやり方を教えて」 → `npx skills find pr review`
- 「変更履歴（changelog）を作成したい」 → `npx skills find changelog`

### ステップ 4: 推奨前に品質を確認する

**検索結果だけで安易に推奨しないでください。** 必ず以下を確認してください：

1. **インストール数** — 1,000回以上のインストールがあるものが好ましいです。100回未満の場合は注意が必要です。
2. **ソースの信頼性** — 公式ソース (`vercel-labs`, `anthropics`, `microsoft` など) は信頼性が高いです。
3. **GitHubスター数** — ソースリポジトリを確認しましょう。100スター未満のリポジトリのスキルは、慎重に扱う必要があります。

### ステップ 5: ユーザーに選択肢を提示する

適切なスキルが見つかったら、以下の情報を提示します：

1. スキル名と機能の概要
2. インストール数とソース
3. インストール用コマンド
4. skills.sh の詳細リンク

回答例：
```
役立ちそうなスキルが見つかりました！ "react-best-practices" スキルは、Vercel Engineering による React と Next.js のパフォーマンス最適化ガイドラインを提供します。（インストール数: 18.5万）

インストールするには：
npx skills add vercel-labs/agent-skills@react-best-practices

詳細はこちら： https://skills.sh/vercel-labs/agent-skills/react-best-practices
```

### ステップ 6: インストールの代行

ユーザーが進めたい場合は、代わりにインストールしてあげることができます：

```bash
npx skills add <owner/repo@skill> -g -y
```

`-g` フラグはグローバル（ユーザーレベル）にインストールし、`-y` は確認プロンプトをスキップします。

## 主要なスキルカテゴリー

検索の際は、以下のカテゴリーを参考にしてください：

| カテゴリー | 検索キーワード例 |
| :--- | :--- |
| Web開発 | react, nextjs, typescript, css, tailwind |
| テスト | testing, jest, playwright, e2e |
| DevOps | deploy, docker, kubernetes, ci-cd |
| ドキュメント | docs, readme, changelog, api-docs |
| コード品質 | review, lint, refactor, best-practices |
| デザイン | ui, ux, design-system, accessibility |
| 生産性 | workflow, automation, git |

## 効果的な検索のコツ

1. **具体的なキーワードを使う**: 単に "testing" とするより "react testing" の方が適切です。
2. **類義語を試す**: "deploy" で見つからなければ "deployment" や "ci-cd" を試してください。
3. **有名ソースをチェック**: 多くのスキルは `vercel-labs/agent-skills` や `ComposioHQ/awesome-claude-skills` から提供されています。

## スキルが見つからない場合

該当するスキルがない場合は：

1. 一致するスキルが見つからなかったことを伝える
2. 通常の機能で直接サポートすることを提案する
3. `npx skills init` で自作スキルを作成することを提案する

例：
```
「xyz」に関連するスキルを検索しましたが、見つかりませんでした。
直接お手伝いすることも可能です。このまま作業を進めますか？

また、定型的な作業であれば、ご自身でスキルを作成することもできます：
npx skills init my-xyz-skill
```
