# 📝 Note Publisher プラグイン

## 目的
記事が完成した後、ブラウザへのコピペ、画像のアップロード、有料エリアの設定などを自動化し、公開ボタンを押すだけ（または完全自動化）の状態にする。

## 仕様

### アプローチ
noteには公式の投稿APIが存在しないため、**ブラウザの自動化（browser_subagent / Playwright）** を使用する。

### 実行フロー
1. **入力受付**: 軍師エージェントから指定されたドラフトファイル（`.md`）およびアイキャッチ画像を受け取る。
2. **ブラウザ起動**: 
   - `https://note.com/editor` へアクセス（要ログイン維持設定済プロファイル）。
3. **タイトルとアイキャッチ挿入**:
   - `#` (MarkdownのH1) をタイトルとして入力。
   - 生成されたアイキャッチ画像をアップロード。
4. **本文の流し込み**:
   - Markdown記述をクリップボード経由で貼り付けるか、リッチテキストエディタに順次ブロック入力。
5. **有料エリア設定**:
   - ドラフト内に `<!-- PREMIUM_ZONE_START -->` などのタグがある場合、そこで「有料ライン」を挿入。
   - 価格を大憲章に基づく竹/梅などの価格に設定（例: 980円）。
6. **保存処理**:
   - 全て完了後、「下書き保存」をクリック。
   - 投稿URL（下書きURL）を王への軍師報告書に記載する。

## 制約と注意点
- サイトのDOM構造変更によりエラーになる場合があるため、失敗時はGuardianがエラーログをキャッチしアラートを上げる。
