# 💸 自動販売ファネル構築ワークフロー

このワークフローは、単発の記事作成を超えて、一度作れば自動で売れ続ける「販売網（ファネル）」をパッケージとして完成させます。

## 📋 事前準備
- [ ] 売りたいコンセプト（例: 「副業初心者向けAI活用術」）を一つ決める。

## 🛠️ ステップ詳細

### 1. ファネル設計と商品企画
// turbo
1. AIに対し、以下の要素を検討させる。
   - バックエンド商品（note）の内容と価格（1,000〜3,000円）
   - フロントエンド（無料・少額）での価値提供
2. 「なぜ今、これが必要なのか」という強力な教育コンセプトを策定する。

### 2. 教育用ステップメールの作成
// turbo
1. LINEやメルマガで配信する「3〜5通のステップメール」を生成する。
   - 1通目: 期待感と自己紹介
   - 2通目: 現状の不安と真実
   - 3通目: 解決策の提示
   - 4通目: 商品案内とベネフィット
   - 5通目: 重大発表・限定性の強調
2. 各メールの「件名」を、開封率が高まるようABテスト案含め生成する。

### 3. 最強セールスレター（LP）の執筆
// turbo
1. noteの販売ページ（またはLP）用のガチなセールスライティングを生成する。
   - PASBECONA（問題・親近感・解決・利益・証拠・限定・行動）の型を適用。
2. 読者が「買わない理由」を一つずつ潰すFAQセクションを追加する。

### 4. 集客ファネルの入り口（SNS）作成
// turbo
1. このファネルに誘導するためのXスレッド、箇条書きポストを量産させる。
2. LINE/メルマガ登録への「無料プレゼント（特典）」の企画と制作。

---
> [!IMPORTANT]
> すべてのステップで「一貫性」を持たせてください。SNSの投稿を読んだ人が、LPに辿り着いた時に「求めていたのはこれだ！」と確信できる動線が理想です。
