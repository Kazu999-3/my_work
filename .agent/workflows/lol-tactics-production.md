# ⚔️ LoL軍師：戦術記事生産ワークフロー

このワークフローは、最新のパッチデータとAIの分析を組み合わせ、プレイヤーが「今すぐ使える」戦術記事を爆速で生成するためのものです。

## 📋 事前準備
- [ ] 公式パッチノート（leagueoflegends.com）から、主要なバフ・ナーフ情報を取得済みであること。
- [ ] `knowledge/domains/lol/` 内に過去の同様の分析ファイルがないか確認する。

## 🛠️ ステップ詳細

### 1. ディープ・メタ・リサーチ
// turbo
1. アンちゃんに対し、特定チャンピオン（例：リリア）のパッチ前後での「具体的数値の差」と、それが「クリア速度・対面性能」にどう影響するか推論させる。
2. 競技シーン（LPL/LCK/LEC等）での採用率を `search_web` で確認し、トレンドの裏付けを取る。

### 2. 「軍師の助言」執筆
// turbo
1. `note-production` ワークフローを呼び出し、以下のLoL特化構成でドラフトを作成する。
   - **導入**: パッチによる環境変化の要約（なぜ今、このチャンピオンなのか）。
   - **ミクロの極意**: 強化数値を活かすための新コンボや新ビルド。
   - **マクロの軍略**: パッチの影響を受けるオブジェクト周りの動き（例：リフレッシュ早まったパッシブをどう使うか）。
   - **有料エリア**: AIだけが気づいた「隠れたTier S」または「アンチピック戦略」。

### 3. ビジュアル & プロモーション
// turbo
1. `generate_image` で、分析対象のチャンピオンをテーマにした、サイバーパンクかつ「軍師」感のあるアイキャッチ画像を生成。
2. X（Twitter）での告知ポスト作成：
   - 1枚目：衝撃的な勝率変化や数値バグ級の強化をアピール。
   - 2枚目以降：記事のサマリー（図解風テキスト）。
   - 最後：noteへのリンク。

## 🎯 期待される成果
- パッチリリースから24時間以内の「最速攻略」という圧倒的な先行者利益。
- 「アンちゃんリサーチ」のブランド化による、リピーターの獲得。
