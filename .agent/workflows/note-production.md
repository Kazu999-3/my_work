# 🚀 収益化note量産ワークフロー（500円モデル）

このワークフローは、Geminiの力を最大限に活用し、安定して読まれる「低単価note」を最速で作成・パッケージングするための手順です。

## 📋 事前準備
- [ ] `knowledge/reports/` 内の最新のトレンドレポート、または `knowledge/memo/` 内の特定のメモを一つ選ぶ。

## 🛠️ ステップ詳細

### 1. 超・ディープリサーチ (Protocol v3)
1. **多角的データ収集**: `31_deep_research_v3.md` に基づき、ターゲットとするテーマを 5 方向から徹底調査。
2. **リサーチ報告書 (Artifact) の作成**: 執筆前に、具体的な数値、クロスリファレンス、独自推論をまとめた報告書を作成し、品質の「核」を確定させる。
3. **承認**: 調査結果の密度に納得してから Step 2 へ移行する。

### 2. 構成案とフック設計
1. リサーチ結果を元に、`30_content_factory.md` の 8 モジュールを含んだ詳細構成を作成。
2. 読者の感情を動かすフック（驚き・共感・好奇心）をタイトルと導入に設計する。

### 3. 本文の「極限」執筆
1. **最低 8,000〜10,000 文字以上**を目指し、各セクションの密度（具体的対面、秒単位の解説、失敗談）を高める。
2. `note_article_drafter.md` を使用し、デザインされた無料・有料の断絶を作る。
3. 生成されたドラフトに対し、`style-auditor` を適用して AI 特有の語彙や曖昧な断定をすべて切除する。

### 4. プロモーション・パッケージング
1. note の表示最適化を考慮したタイトル（32文字以内）の磨き込み。
2. **連載型 X スレッド (4〜6連投)** を生成し、記事の一部を有益情報としてギブするプロモーション体制を構築。
3. `generate_image` を用い、記事の価値を象徴するプレミアムなアイキャッチを生成。

## 🎯 期待される成果
- 読者が「500円でも安すぎる」と感じる、圧倒的情報密度の実用コンテンツ。
- 公開初日から X での拡散と購入を同時に狙える、宣伝素材を含めたパッケージ。

---
> [!IMPORTANT]
> 「量より質、そして質の圧倒的な量（文字数と密度）」が Antigravity の収益化モデルの基盤です。リサーチの妥協は記事の死を意味すると心得よ。
