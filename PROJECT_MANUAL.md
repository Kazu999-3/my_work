# 🌍 Antigravity プロジェクト全体マニュアル

本ディレクトリ（`d:\my_work`）は、AIアシスタント「アンちゃん」が管理する、自動化ツールと知識の集積地です。
このドキュメントでは、現在稼働している各システムと、それらを支えるスキルの全体像を解説します。

---

## 🏗️ 1. 主要アプリケーション一覧 (Apps)

各アプリは `apps/` フォルダに格納されており、個別の目的を持って動作しています。

### 🤖 [Hybrid Bot (Discord x Notion)](file:///d:/my_work/apps/hybrid_bot)
- **役割**: 日常のタスク管理、アイデア要約、知識ベース対話。
- **機能**: 
  - Discordからのタスク登録（期日解析）と完了操作。
  - URLによるWebサイト要約（X/Twitter対応）とNotionメモ帳への自動保存。
  - 過去のメモやLoL知識に基づいた対話（RAG機能：`/ask`, `/ask-lol`）。
- **主要コマンド**: `/memo`, `/task`, `/tasks`, `/done`, `/ask`, `/ask-lol`, `/review`, `/ban`, `/draft`, `/sync`

### ✍️ [Note Generator](file:///d:/my_work/apps/note_generator)
- **役割**: トレンド分析から記事作成・画像生成までを自動化。
- **機能**: 
  - トレンドキーワードの取得と分析。
  - 構成案の自動生成と本文執筆。
  - 収益化の観点からの自動レビュー。

### 📊 [YouTube Manager](file:///d:/my_work/apps/youtube_manager)
- **役割**: YouTubeプレイリストの自動整理と在庫・履歴管理。
- **使い方**: 
  1. Notionの「YouTube整理ルール (Rules)」DBに移動。
  2. チャンネル名やキーワード、移動先の再生リスト名を登録。
  3. YouTube上で整理したい動画を「整理前」プレイリストに入れる。
  4. `tools\RUN_YT_CLEAN.bat` を実行。
- **機能**: 
  - Notionのルールに基づく自動移動・削除（「ジャンル移動」「自動削除」）。
  - 削除済み・非公開動画の自動クリーニング。
  - Notion上で「動画在庫 (Inventory)」と「実行履歴 (Log)」を一元管理。

## 📂 4. ディレクトリ構成の要約

- `apps/`: 実装されたプログラム本体。
- `tools/`: **[NEW]** 各種自動化・同期用バッチファイル（.bat）の格納場所。
- `knowledge/memo/`: Notionから同期したメモ（Markdown）の蓄積場所。
- `outputs/`: AIによって生成された記事や成果物の保存場所。
- `skills/`: AIの思考ロジック（専門知識）。
- `workflows/`: 一連の作業をパッケージ化した手順。
- `rules/`: プロジェクト全体で守るべき「憲法」。

## 🚀 バッチマニュアル (tools/)

### 🕒 【定期実行・常用】
- `tools\RUN_YT_CLEAN.bat`: YouTubeの整理。Notionでルールを変えた時や「整理前」に動画を溜めた時に実行。
- `tools\SYNC_ALL.bat`: **[最重要]** 全同期・個別同期の統合メニュー。これ一つでタスク、文書、メモ、YouTubeのすべての操作が可能です。

### 🛠️ 【初期設定・保守】
- `tools\RUN_BOT.bat`: **[必須]** Discord Botの起動。PC起動時に一回実行してください。

## ⚙️ 5. システム領域（移動・削除禁止）

以下のフォルダはシステムの動作に不可欠な「裏方」です。通常、ユーザーが中身を直接編集したり、場所を移動させる必要はありません。

- `.venv/`: **Python仮想環境**。同期スクリプトやYouTube Manager等の実行に必要なライブラリが格納されています。
- `.vscode/`: **エディタ設定**。Visual Studio Codeで快適に作業（コードの色分け、自動整形、Pythonの紐付け等）を行うための設定ファイル群です。
